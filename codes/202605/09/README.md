# AI 기반 텍스트 요약기 프로젝트

이 프로젝트는 AI 기반 텍스트 요약 서비스의 소규모 예시를 보여주는 자바 애플리케이션입니다. 사용자로부터 텍스트를 입력받아 이를 요약된 형태로 출력하는 기능을 제공합니다. 실제 AI 모델과의 연동 대신, 간략화된 로직을 사용하여 AI 요약기의 개념을 시뮬레이션합니다.

## 프로젝트 구조

- `Main.java`: 애플리케이션의 시작점입니다. 사용자 입력을 처리하고 요약 서비스를 호출하여 결과를 출력합니다.
- `SummarizationService.java`: 텍스트 요약 기능을 정의하는 인터페이스입니다. 다양한 요약 구현체를 위한 계약을 제공합니다.
- `SimpleTextSummarizer.java`: `SummarizationService` 인터페이스의 구현체입니다. 실제 AI 모델 대신, 간단한 규칙 기반의 요약 로직을 포함합니다. (예: 텍스트의 첫 부분을 추출)
- `ConsoleUtil.java`: 콘솔 입출력을 위한 유틸리티 메서드를 제공합니다.

## 실행 방법

1.  **컴파일:**
    프로젝트 디렉터리로 이동하여 다음 명령어를 사용하여 자바 소스 파일을 컴파일합니다.
    ```bash
    javac com/gemini/summarizer/*.java
    ```
2.  **실행:**
    컴파일이 완료되면 다음 명령어를 사용하여 애플리케이션을 실행합니다.
    ```bash
    java com.gemini.summarizer.Main
    ```
3.  **사용:**
    애플리케이션이 시작되면 텍스트를 입력하라는 메시지가 표시됩니다. 요약하고자 하는 텍스트를 입력하고 엔터를 누르면, 요약된 결과가 출력됩니다.
