# JavaScript 텍스트 요약 도구

이 프로젝트는 주어진 텍스트를 요약하는 간단한 JavaScript 도구이다. 핵심 문장 추출 방식을 사용하여 텍스트에서 가장 중요한 문장들을 식별하고 추출한다.

## 주요 기능

*   텍스트를 문장 단위로 분리한다.
*   각 문장의 중요도를 평가한다.
*   가장 중요한 문장들을 추출하여 요약문을 생성한다.

## 코드 구조

*   `main.js`: 프로그램의 진입점이다. 예제 텍스트를 로드하고 `Summarizer`를 사용하여 요약문을 생성한 후 출력한다.
*   `summarizer.js`: 텍스트 요약의 핵심 로직을 포함하고 있다. 문장 중요도 계산 및 요약문 생성 기능을 제공한다.
*   `textUtils.js`: 텍스트 처리 유틸리티 함수들을 모아 놓았다. 문장 분리, 불용어 제거 등의 기능을 담당한다.

## 실행 방법

1.  Node.js가 설치되어 있는지 확인한다.
2.  프로젝트 디렉토리로 이동한다.
    ```bash
    cd C:\server\gpt-code-diary\codes\202605\02
    ```
3.  다음 명령어를 실행하여 요약 도구를 실행한다.
    ```bash
    node main.js
    ```
4.  콘솔에 요약된 텍스트가 출력될 것이다.

## 사용된 기술

*   **JavaScript (Node.js 환경)**: 서버 측 JavaScript 런타임 환경이다.
*   **간단한 핵심 문장 추출 알고리즘**: 단어 빈도수를 기반으로 문장의 중요도를 평가하는 방법을 사용한다. (실제 복잡한 NLP 라이브러리 사용을 가정하였으나, 구현은 단순화된 형태로 제공된다.)

## 기여

이 프로젝트는 간단한 예시를 위한 것이며, 실제 사용을 위한 고급 요약 기능은 포함하지 않는다. 기능 개선 제안이나 버그 수정은 환영한다.
