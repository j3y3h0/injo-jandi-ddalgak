# 간단한 상품 리뷰 감성 분석기

## 주제
AI 및 데이터 분석을 활용한 시장 동향 예측

## 프로젝트 설명
이 프로젝트는 주어진 상품 리뷰 데이터를 분석하여 긍정, 부정, 중립으로 분류하는 간단한 JavaScript 애플리케이션입니다. 이는 대량의 텍스트 데이터에서 시장의 전반적인 감성이나 트렌드를 파악하는 기초적인 단계로 활용될 수 있습니다. 예를 들어, 특정 상품에 대한 소비자들의 반응을 빠르게 요약하여 마케팅 전략 수립에 도움을 줄 수 있습니다.

## 파일 구성
*   `main.js`: 애플리케이션의 진입점입니다. `dataProcessor.js` 모듈을 사용하여 `data.js`에 정의된 샘플 리뷰 데이터를 처리하고, 그 결과를 콘솔에 출력합니다.
*   `dataProcessor.js`: 핵심 감성 분석 로직을 포함하고 있습니다. 상품 리뷰 텍스트를 분석하여 미리 정의된 키워드 목록을 기반으로 감성 점수를 계산하고 최종적으로 "긍정", "부정", "중립"으로 분류합니다.
*   `data.js`: 감성 분석에 사용될 샘플 상품 리뷰 데이터가 배열 형태로 정의되어 있습니다. 각 리뷰는 ID와 텍스트 내용을 포함합니다.

## 실행 방법
이 프로젝트는 Node.js 환경에서 실행됩니다. Node.js가 설치되어 있지 않다면 먼저 설치해야 합니다. 프로젝트를 실행하려면 터미널에서 다음 명령어를 입력하십시오:

```bash
node main.js
```

명령어를 실행하면 각 리뷰에 대한 감성 분석 결과가 콘솔에 출력될 것입니다.
