# C# 감정 분석 도구 (Sentiment Analysis Tool)

## 프로젝트 개요

이 프로젝트는 입력된 텍스트의 감정을 분석하여 긍정, 부정 또는 중립으로 분류하는 간단한 C# 감정 분석 도구입니다. 이 도구는 텍스트 처리의 기본 개념을 활용하여 AI 에이전트의 초기 단계를 시뮬레이션하며, 고객 피드백 분석, 소셜 미디어 모니터링 등 다양한 산업 분야에서 활용될 수 있는 감정 분석의 기초를 제공합니다.

## 포함된 파일

*   `Program.cs`: 프로그램의 진입점입니다. 사용자로부터 텍스트를 입력받고, `SentimentAnalyzer`를 호출하여 분석을 수행한 후 결과를 출력합니다.
*   `SentimentAnalyzer.cs`: 감정 분석의 핵심 로직을 포함합니다. `Lexicon` 클래스를 사용하여 텍스트의 감정 점수를 계산합니다.
*   `Lexicon.cs`: 감정 분석에 사용될 긍정 및 부정 단어 목록(어휘집)을 정의합니다.

## 코드 구조 설명

이 프로젝트는 세 가지 주요 클래스로 구성되어 있습니다.

1.  **`Program` 클래스**:
    *   `Main` 메서드는 사용자 콘솔 입력을 처리합니다.
    *   `SentimentAnalyzer` 인스턴스를 생성하고, 입력된 텍스트에 대해 `Analyze` 메서드를 호출합니다.
    *   분석 결과를 콘솔에 출력합니다.

2.  **`SentimentAnalyzer` 클래스**:
    *   텍스트를 단어 단위로 분리하고, 각 단어가 `Lexicon`에 정의된 긍정 또는 부정 단어인지 확인합니다.
    *   긍정 단어에 대해서는 점수를 추가하고, 부정 단어에 대해서는 점수를 차감하여 총 감정 점수를 계산합니다.
    *   최종 점수를 기반으로 감정(`Positive`, `Negative`, `Neutral`)을 결정합니다.

3.  **`Lexicon` 클래스**:
    *   `PositiveWords` 및 `NegativeWords`라는 두 개의 `HashSet<string>` 컬렉션을 포함합니다.
    *   이 컬렉션에는 감정 분석에 사용될 미리 정의된 긍정적 및 부정적 단어들이 저장되어 있습니다.

## 실행 방법

이 프로젝트를 실행하려면 .NET SDK가 설치되어 있어야 합니다.

1.  **프로젝트 복제 및 이동 (현재 작업 디렉터리에 파일들이 이미 생성되었으므로 해당하지 않음)**:
    이 파일들은 이미 `C:\server\gpt-code-diary\codes\202605\10` 경로에 생성되었습니다.

2.  **빌드**:
    명령 프롬프트 또는 터미널을 열고 프로젝트 디렉터리(`C:\server\gpt-code-diary\codes\202605\10`)로 이동한 다음 다음 명령을 실행합니다:
    ```bash
    dotnet build
    ```

3.  **실행**:
    빌드가 성공적으로 완료되면 다음 명령을 사용하여 프로그램을 실행할 수 있습니다:
    ```bash
    dotnet run
    ```
    프로그램이 실행되면 텍스트를 입력하라는 메시지가 표시됩니다. 텍스트를 입력하고 Enter 키를 누르면 감정 분석 결과가 출력됩니다.

## 예시

```
텍스트를 입력하세요: 이 영화는 정말 최고였어요! 배우들의 연기도 훌륭하고 스토리는 완벽했어요.
감정: Positive

텍스트를 입력하세요: 서비스가 너무 느리고 실망스러웠습니다. 다시는 이용하지 않을 것입니다.
감정: Negative

텍스트를 입력하세요: 날씨가 좋네요. 평범한 하루였습니다.
감정: Neutral
```
