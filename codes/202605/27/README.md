# 간단한 판매 데이터 분석 및 시각화 프로젝트

## 프로젝트 개요
이 프로젝트는 특정 기간 동안의 가상 판매 데이터를 분석하고, 그 결과를 텍스트 기반 차트로 시각화하는 자바스크립트 애플리케이션이다. 실제 산업 환경에서는 D3.js나 Chart.js와 같은 전문 시각화 라이브러리가 사용되지만, 이 프로젝트는 순수 자바스크립트만을 사용하여 데이터 처리 및 기본적인 시각화 로직을 보여주는 데 중점을 둔다. 이는 백엔드 환경이나 경량 환경에서의 데이터 요약 및 보고에 유용할 수 있다.

## 코드 구조
프로젝트는 다음과 같은 파일들로 구성되어 있다:

*   **`data.js`**:
    *   분석할 원본 판매 데이터를 포함하는 모듈이다. 일반적으로 데이터베이스나 외부 API로부터 로드될 수 있는 형식을 모방하여 자바스크립트 배열 객체 형태로 정의되어 있다.
*   **`analyzer.js`**:
    *   `data.js`에서 제공된 데이터를 분석하고 유의미한 통계 또는 요약 정보를 추출하는 함수들을 정의한다. 예를 들어, 카테고리별 총 판매액 계산과 같은 기능을 수행한다.
*   **`visualizer.js`**:
    *   `analyzer.js`에서 분석된 데이터를 콘솔에 출력하기 적합한 텍스트 기반의 시각적 형태로 변환하는 함수들을 정의한다. 막대 차트나 요약 테이블 등을 생성할 수 있다.
*   **`main.js`**:
    *   애플리케이션의 주요 진입점이다. `data.js`에서 데이터를 불러오고, `analyzer.js`를 사용하여 데이터를 분석하며, 마지막으로 `visualizer.js`를 통해 분석 결과를 콘솔에 출력하는 과정을 총괄한다.

## 실행 방법
프로젝트를 실행하려면 Node.js 환경이 필요하다. 다음 명령어를 사용하여 `main.js` 파일을 실행할 수 있다.

```bash
node main.js
```

이 명령어를 실행하면 `data.js`에 포함된 판매 데이터가 분석되고, 그 결과가 텍스트 기반의 시각화 형태로 콘솔에 출력될 것이다.
