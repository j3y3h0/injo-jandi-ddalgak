# 데이터 분석 프로젝트: 기본 통계 계산기

## 프로젝트 개요
이 프로젝트는 주어진 숫자 데이터 배열에 대한 기본적인 통계(합계, 평균, 중앙값, 표준편차)를 계산하는 간단한 자바 애플리케이션이다. 데이터 처리 및 통계 계산의 기초적인 예시를 제공하며, 산업에서 활용될 수 있는 데이터 분석 라이브러리의 핵심 기능을 소규모로 구현해 본다.

## 코드 구조
프로젝트는 다음과 같은 자바 파일들로 구성되어 있다:

1.  `Main.java`: 애플리케이션의 시작점이다. 샘플 데이터를 생성하고, `DataProcessor`와 `StatisticsUtil` 클래스를 사용하여 데이터를 처리하고 통계를 계산한 후 결과를 출력한다.
2.  `DataProcessor.java`: 데이터 전처리(예: 데이터 합계 계산)를 담당하는 유틸리티 클래스이다.
3.  `StatisticsUtil.java`: 숫자 배열에 대한 평균, 중앙값, 표준편차와 같은 통계적 계산을 수행하는 유틸리티 클래스이다.

## 실행 방법
이 프로젝트를 실행하기 위해서는 자바 개발 키트(JDK) 8 이상이 설치되어 있어야 한다.

1.  **컴파일:** 터미널을 열고 프로젝트 디렉터리(`C:\server\gpt-code-diary\codes\202605\20`)로 이동한 후 다음 명령어를 실행한다.
    ```bash
    javac Main.java DataProcessor.java StatisticsUtil.java
    ```
2.  **실행:** 컴파일이 성공적으로 완료되면 다음 명령어를 사용하여 애플리케이션을 실행한다.
    ```bash
    java Main
    ```

애플리케이션은 콘솔에 샘플 데이터에 대한 통계 분석 결과를 출력할 것이다.

## 사용된 산업 라이브러리/알고리즘 (개념적 적용)
이 프로젝트는 외부 라이브러리 없이 순수 자바 코드로 구현되었지만, 다음과 같은 개념들을 포함한다:

*   **데이터 전처리:** `DataProcessor`에서 데이터의 합계를 계산하는 등 데이터 분석의 첫 단계를 모방한다.
*   **기본 통계 알고리즘:** 평균, 중앙값, 표준편차 계산은 데이터 분석 분야에서 가장 기본적이고 널리 사용되는 통계 알고리즘이다. `StatisticsUtil` 클래스에서 이러한 알고리즘들을 직접 구현하여 그 원리를 이해할 수 있도록 하였다. 실제 산업 환경에서는 Apache Commons Math와 같은 강력한 통계 라이브러리를 활용하여 더욱 복잡하고 최적화된 계산을 수행한다.
