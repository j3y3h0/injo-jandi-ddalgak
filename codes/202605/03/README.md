# 간단한 텍스트 감성 분석기

이 프로젝트는 간단한 Java 기반의 텍스트 감성 분석기이다. 미리 정의된 긍정/부정 단어 사전을 이용하여 입력된 텍스트의 감성(긍정, 부정, 중립)을 분류한다.

## 프로젝트 구조

```
.
├── Main.java
├── SentimentAnalyzer.java
└── SentimentLexicon.java
```

*   `Main.java`: 프로그램의 진입점이다. `SentimentLexicon`과 `SentimentAnalyzer`를 사용하여 예시 문장들의 감성을 분석하고 결과를 출력한다.
*   `SentimentAnalyzer.java`: 텍스트를 분석하여 감성 점수를 계산하고, 이를 기반으로 감성 레이블("긍정", "부정", "중립")을 반환하는 핵심 로직을 포함한다.
*   `SentimentLexicon.java`: 감성 분석에 사용되는 긍정적인 단어들과 부정적인 단어들을 저장하고 관리하는 클래스이다.

## 동작 방식

이 감성 분석기는 어휘 기반 접근 방식을 사용한다.
1.  `SentimentLexicon`에 정의된 긍정 및 부정 단어 목록을 로드한다.
2.  `SentimentAnalyzer`는 입력된 텍스트를 단어 단위로 분리한다. (현재는 간단한 공백 및 특수문자 제거 후 단어 추출 방식이며, 실제 산업에서는 형태소 분석기 사용이 일반적이다.)
3.  분리된 각 단어가 긍정 단어 목록에 있으면 감성 점수를 증가시키고, 부정 단어 목록에 있으면 감성 점수를 감소시킨다.
4.  최종 감성 점수에 따라 텍스트의 감성을 "긍정", "부정", "중립"으로 분류한다.

## 실행 방법

이 프로젝트는 Java Development Kit (JDK) 8 이상이 설치되어 있어야 한다.

1.  **컴파일:** 현재 작업 디렉터리에서 다음 명령어를 실행하여 Java 소스 코드들을 컴파일한다.

    ```bash
    javac Main.java SentimentAnalyzer.java SentimentLexicon.java
    ```

    또는 모든 `.java` 파일을 한 번에 컴파일할 수도 있다.

    ```bash
    javac *.java
    ```

2.  **실행:** 컴파일이 성공적으로 완료되면, 다음 명령어를 실행하여 프로그램을 실행할 수 있다.

    ```bash
    java Main
    ```

    프로그램은 미리 정의된 예시 문장들의 감성 분석 결과를 콘솔에 출력할 것이다.

## 확장 계획 (선택 사항)

*   **감성 사전 확장:** `SentimentLexicon.java`에 더 많은 단어를 추가하거나, 외부 파일(CSV, JSON 등)에서 단어를 로드하도록 구현할 수 있다.
*   **형태소 분석기 통합:** 한국어 텍스트의 정확한 분석을 위해 KoNLP, Hannanum 등과 같은 형태소 분석 라이브러리를 통합하여 단어 분리 정확도를 높일 수 있다.
*   **머신러닝 모델 적용:** Naive Bayes, Support Vector Machine (SVM) 또는 딥러닝(RNN, LSTM) 기반의 감성 분석 모델을 학습하고 통합하여 더 정교한 분석을 수행할 수 있다.
*   **가중치 부여:** 단어의 중요도에 따라 다른 가중치를 부여하여 감성 점수 계산의 정확도를 높일 수 있다.
