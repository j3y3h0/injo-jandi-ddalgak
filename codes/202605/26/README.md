# 주식 시장 데이터 분석 및 예측 시뮬레이션 프로젝트

이 프로젝트는 가상의 주식 시장 데이터를 로드하고, 기술적 분석(이동 평균)을 수행하여 간단한 투자 권고를 제공하는 시뮬레이션입니다. JavaScript를 사용하여 모듈화된 형태로 구현되었으며, 실제 시장 데이터 분석 시스템의 기초적인 구조를 이해하는 데 도움을 줍니다.

## 1. 프로젝트 목적

*   주식 데이터 처리 및 분석의 기본 개념 이해
*   이동 평균과 같은 기술적 지표 계산 방법 학습
*   간단한 투자 전략 구현 및 시뮬레이션
*   JavaScript 모듈 시스템(`import`/`export`) 활용

## 2. 코드 구조 설명

프로젝트는 다음과 같은 파일들로 구성되어 있습니다:

*   `main.js`:
    *   이 프로젝트의 주 실행 파일이다.
    *   `dataLoader`, `analysisService`, `reportGenerator` 모듈을 가져와 전체 데이터 흐름을 제어한다.
    *   데이터 로드, 분석, 권고 생성 및 보고서 출력 과정을 순차적으로 실행한다.

*   `dataLoader.js`:
    *   주식 데이터를 로드하는 역할을 담당한다.
    *   현재는 고정된 모의 데이터를 비동기적으로 제공하지만, 실제 환경에서는 외부 API 또는 데이터베이스에서 데이터를 가져오도록 확장될 수 있다.
    *   데이터 형식은 `[{ date: "YYYY-MM-DD", price: 숫자 }]` 이다.

*   `analysisService.js`:
    *   로드된 주식 데이터에 대한 기술적 분석을 수행한다.
    *   주요 기능은 지정된 기간(예: 5일)의 이동 평균을 계산하는 것이다.
    *   계산된 지표를 바탕으로 현재 종가와 이동 평균을 비교하여 '매수', '매도', '관망'과 같은 간단한 투자 권고를 생성한다.

*   `reportGenerator.js`:
    *   `analysisService`에서 생성된 분석 결과와 투자 권고를 보기 쉽고 이해하기 쉬운 형식으로 콘솔에 출력한다.
    *   최근 데이터 요약 및 최종 투자 권고를 포함한 보고서를 생성한다.

## 3. 실행 방법

이 프로젝트를 실행하려면 Node.js가 설치되어 있어야 한다.

1.  **Node.js 설치**: [Node.js 공식 웹사이트](https://nodejs.org/)에서 Node.js를 다운로드하여 설치한다.
2.  **프로젝트 디렉터리로 이동**: 터미널 또는 명령 프롬프트에서 이 프로젝트 파일들이 있는 디렉터리로 이동한다.
    ```bash
    cd C:\server\gpt-code-diary\codes\202605\26
    ```
3.  **프로젝트 실행**: 다음 명령어를 사용하여 `main.js` 파일을 실행한다.
    ```bash
    node main.js
    ```

### 예상 출력

프로그램을 실행하면 데이터 로드, 분석 진행 상황 및 최종 분석 보고서가 콘솔에 출력될 것이다.

---
**참고**: 이 프로젝트는 교육 및 시뮬레이션 목적으로만 사용해야 한다. 실제 투자 결정에는 활용할 수 없으며, 실제 시장 데이터는 훨씬 복잡하고 다양한 요인을 고려해야 한다.
