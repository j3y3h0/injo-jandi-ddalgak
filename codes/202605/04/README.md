# 간단한 AI 챗봇 프로젝트

이 프로젝트는 미리 정의된 패턴과 응답을 사용하여 사용자 질문에 답하는 간단한 커맨드라인 챗봇을 JavaScript로 구현한 것이다.

## 프로젝트 구조

*   `main.js`: 챗봇 애플리케이션의 메인 진입점이다. 사용자 입력을 처리하고 챗봇 로직을 호출하여 응답을 출력한다.
*   `chatbot.js`: 챗봇의 핵심 로직을 포함한다. `data.js`에 정의된 패턴과 사용자 입력을 비교하여 적절한 응답을 선택한다.
*   `data.js`: 챗봇이 사용하는 응답 데이터를 포함하는 파일이다. 각 객체는 `pattern` (정규 표현식)과 `responses` (응답 메시지 배열)로 구성된다.
*   `package.json`: 프로젝트의 메타데이터와 의존성 라이브러리(`readline-sync`) 정보를 담고 있다.

## 실행 방법

프로젝트를 실행하려면 다음 단계를 따른다.

1.  **의존성 설치**: 프로젝트 폴더에서 터미널을 열고 다음 명령어를 실행하여 필요한 라이브러리를 설치한다.
    ```bash
    npm install
    ```
2.  **챗봇 실행**: 의존성 설치가 완료되면 다음 명령어를 사용하여 챗봇을 시작한다.
    ```bash
    node main.js
    ```
3.  **대화 시작**: 챗봇이 시작되면 프롬프트(`당신: `)에 질문을 입력하여 챗봇과 대화할 수 있다.
4.  **종료**: 챗봇을 종료하려면 `종료`라고 입력한다.

## 사용된 기술

*   **JavaScript**: 프로젝트의 주요 개발 언어이다.
*   **Node.js**: JavaScript 런타임 환경이다.
*   **readline-sync**: 사용자 입력을 동기적으로 받을 수 있도록 하는 Node.js 라이브러리이다.

이 챗봇은 간단한 패턴 매칭 기반으로 작동하며, 복잡한 자연어 처리 기능은 포함되어 있지 않다. 새로운 패턴과 응답을 `data.js` 파일에 추가하여 챗봇의 기능을 확장할 수 있다.
