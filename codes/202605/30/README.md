# 파이썬 기반 감성 분석 프로젝트

이 프로젝트는 파이썬의 `TextBlob` 라이브러리를 활용하여 텍스트의 감성을 분석하는 소규모 애플리케이션이다. 주어진 텍스트가 긍정적인지, 부정적인지, 또는 중립적인지를 판단하는 기능을 제공한다.

## 프로젝트 구조

*   `README.md`: 현재 문서이다. 프로젝트에 대한 설명과 실행 방법을 담고 있다.
*   `main.py`: 프로젝트의 진입점이다. 샘플 텍스트를 사용하여 감성 분석을 실행하고 결과를 출력한다.
*   `sentiment_analyzer.py`: `TextBlob` 라이브러리를 이용하여 실제 감성 분석을 수행하는 로직이 구현되어 있다.
*   `utils.py`: 텍스트 전처리와 관련된 유틸리티 함수들을 포함한다. (현재는 간단한 예시만 포함될 것이다.)

## 주요 기술 스택

*   **언어**: Python 3.x
*   **라이브러리**: `TextBlob` (감성 분석)

## 설치 방법

프로젝트를 실행하기 전에 필요한 라이브러리를 설치해야 한다. 다음 명령어를 사용하여 설치한다:

```bash
pip install TextBlob
python -m textblob.download_corpora
```

`textblob.download_corpora` 명령은 `TextBlob`이 감성 분석을 위해 사용하는 NLTK 데이터셋을 다운로드한다.

## 실행 방법

모든 의존성 설치가 완료되면, `main.py` 파일을 실행하여 감성 분석을 수행할 수 있다.

```bash
python main.py
```

`main.py` 파일 내부에 있는 `sample_texts` 리스트의 내용을 수정하여 다른 텍스트에 대한 감성 분석을 시도할 수 있다.

## 감성 분석 결과 해석

`TextBlob`의 감성 분석 결과는 주로 두 가지 지표를 제공한다:

*   **Polarity (극성)**: -1.0 (매우 부정적)부터 +1.0 (매우 긍정적) 사이의 값을 가진다. 0에 가까울수록 중립적이다.
*   **Subjectivity (주관성)**: 0.0 (매우 객관적)부터 +1.0 (매우 주관적) 사이의 값을 가진다.

이 프로젝트에서는 Polarity 값을 기준으로 텍스트의 감성을 긍정, 부정, 중립으로 분류한다.
    - Polarity > 0: 긍정
    - Polarity < 0: 부정
    - Polarity = 0: 중립
