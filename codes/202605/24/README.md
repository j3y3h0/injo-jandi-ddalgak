# 사용자 리뷰 감성 분석기

## 1. 프로젝트 개요

이 프로젝트는 사용자 리뷰 텍스트를 분석하여 해당 리뷰의 감성(긍정, 부정, 중립)을 판별하는 간단한 애플리케이션이다. 최근 AI 및 머신러닝 기술이 텍스트 처리 분야에서 활발히 사용되는 트렌드를 반영하여, 실용적인 감성 분석 기능을 자바스크립트로 구현하였다. 이 예시는 대규모 데이터 분석 및 고객 피드백 시스템의 기초적인 구성 요소를 보여준다.

## 2. 파일 구조

*   `package.json`: 프로젝트의 메타데이터와 의존성 라이브러리(`natural` 라이브러리는 한국어 분석에 직접 사용되지는 않지만, 다른 NLP 작업에서 활용될 수 있음을 보여주기 위해 포함되었다)를 정의한다.
*   `dataLoader.js`: 감성 분석에 사용될 샘플 사용자 리뷰 데이터를 제공하는 모듈이다. 실제 환경에서는 데이터베이스나 외부 API로부터 데이터를 로드할 수 있다.
*   `sentimentAnalyzer.js`: 텍스트 감성을 분석하는 핵심 로직을 포함한다. 본 예시에서는 복잡한 한국어 NLP 모델 대신 간단한 키워드 매칭 방식을 사용하였으나, 주석으로 더 고급 AI/ML 모델을 통합할 수 있는 가능성을 명시하였다.
*   `main.js`: 애플리케이션의 메인 엔트리 포인트이다. `dataLoader`에서 리뷰를 가져와 `sentimentAnalyzer`로 감성을 분석하고 그 결과를 콘솔에 출력한다.

## 3. 실행 방법

이 프로젝트를 실행하기 위해서는 Node.js 환경이 필요하다.

1.  **의존성 설치**: 프로젝트 디렉터리에서 아래 명령어를 실행하여 필요한 라이브러리를 설치한다.
    ```bash
    npm install
    ```
2.  **애플리케이션 실행**: 설치가 완료되면 다음 명령어로 감성 분석 애플리케이션을 실행한다.
    ```bash
    npm start
    # 또는
    # node main.js
    ```

명령어를 실행하면 샘플 리뷰들이 순차적으로 감성 분석되어 콘솔에 결과가 출력될 것이다.

## 4. 기술 스택

*   **언어**: JavaScript (Node.js 환경)
*   **라이브러리**: `natural` (참고용. 직접적인 한국어 감성 분석에는 사용되지 않음)

## 5. 향후 개선 방향 (실제 산업 적용 시)

*   **고급 NLP 모델 도입**: 한국어에 특화된 사전 훈련된 딥러닝 모델 (예: KoBERT, GPT-3.5/4)을 사용하여 감성 분석의 정확도를 크게 향상시킬 수 있다.
*   **확장 가능한 데이터 로딩**: 실제 서비스에서는 데이터베이스(MongoDB, PostgreSQL 등) 또는 클라우드 스토리지(AWS S3, Google Cloud Storage)에서 대규모 리뷰 데이터를 효율적으로 로드하는 시스템을 구축한다.
*   **웹 인터페이스**: 분석 결과를 시각적으로 보여주거나 사용자가 직접 텍스트를 입력하여 감성을 분석할 수 있는 웹 기반 UI(React, Vue, Angular 등)를 개발할 수 있다.
*   **API 연동**: 감성 분석 기능을 RESTful API로 제공하여 다른 서비스나 애플리케이션에서 쉽게 활용할 수 있도록 한다.
