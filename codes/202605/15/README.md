# 간단한 감성 분석 프로젝트 (Java)

이 프로젝트는 주어진 텍스트의 감성(긍정, 부정, 중립)을 분석하는 간단한 Java 애플리케이션이다. 특정 키워드 목록을 기반으로 텍스트 내 긍정 및 부정 단어의 출현 빈도를 계산하여 감성을 판단한다. 이는 온라인 리뷰, 소셜 미디어 게시물 등에서 전반적인 여론을 파악하는 데 활용될 수 있다.

## 프로젝트 구조

*   `Main.java`: 프로그램의 진입점이다. `SentimentAnalyzer`를 사용하여 여러 예시 텍스트의 감성을 분석하고 결과를 콘솔에 출력한다.
*   `SentimentAnalyzer.java`: 감성 분석의 핵심 로직을 포함한다. 긍정 단어와 부정 단어의 내부 목록을 가지고 있으며, 입력된 텍스트를 분석하여 `Sentiment` enum 값을 반환한다.
*   `Sentiment.java`: 감성 유형(POSITIVE, NEGATIVE, NEUTRAL)을 정의하는 enum이다.

## 실행 방법

이 프로젝트는 표준 Java 개발 키트(JDK)를 사용하여 컴파일하고 실행할 수 있다.

1.  **소스 코드 다운로드:**
    모든 `.java` 파일을 로컬 디렉터리(예: `sentiment_project`)에 저장한다.

2.  **컴파일:**
    명령 프롬프트 또는 터미널을 열고 `.java` 파일이 있는 디렉터리로 이동한다. 다음 명령어를 사용하여 소스 파일을 컴파일한다.

    ```bash
    javac Main.java Sentiment.java SentimentAnalyzer.java
    ```

    또는 모든 Java 파일을 한 번에 컴파일할 수 있다.

    ```bash
    javac *.java
    ```

3.  **실행:**
    컴파일이 성공적으로 완료되면 `.class` 파일이 생성된다. 다음 명령어를 사용하여 프로그램을 실행한다.

    ```bash
    java Main
    ```

## 코드 설명

`SentimentAnalyzer.java` 파일은 내부적으로 `POSITIVE_WORDS`와 `NEGATIVE_WORDS`라는 두 개의 `HashSet`을 유지한다. 이 집합들은 각각 긍정적 또는 부정적 감성을 나타내는 미리 정의된 단어들을 포함한다. `analyze` 메서드는 입력된 텍스트를 받아 소문자로 변환하고, 공백 및 구두점을 기준으로 단어를 분리한다. 분리된 각 단어에 대해 긍정 또는 부정 단어 집합에 포함되는지 확인하여 각각의 카운트를 증가시킨다. 최종적으로 긍정 단어 카운트와 부정 단어 카운트를 비교하여 전반적인 감성을 결정하고 `Sentiment` enum 값을 반환한다. 동일한 개수일 경우 `NEUTRAL`을 반환한다.

## 활용 가능성

이 간단한 모델은 기본적인 감성 분석의 원리를 이해하는 데 도움이 된다. 실제 산업 환경에서는 더 정교한 자연어 처리(NLP) 기술, 머신러닝 모델(예: 텍스트 분류), 그리고 방대한 양의 훈련 데이터셋을 사용하여 정확도를 높인다. 이 프로젝트는 이러한 복잡한 시스템의 기초 단계로 볼 수 있다.
