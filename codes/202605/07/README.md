# 주식 시장 데이터 분석 프로젝트

이 프로젝트는 지정된 주식 심볼에 대한 과거 데이터를 가져와 이동 평균을 계산하고 출력하는 간단한 자바스크립트 애플리케이션이다. `axios` 라이브러리를 사용하여 외부 API에서 데이터를 효율적으로 가져오고, 계산된 이동 평균을 통해 기본적인 시장 동향 분석을 시도한다.

## 파일 구조

*   `main.js`: 애플리케이션의 진입점이다. 주식 데이터를 가져오고 분석하는 전체 흐름을 제어한다.
*   `stock_api.js`: 주식 시장 데이터 API와 통신하는 기능을 담당한다. 특정 주식 심볼에 대한 과거 데이터를 요청하고 받아오는 로직을 포함한다.
*   `analyzer.js`: 가져온 주식 데이터를 분석하는 기능을 제공한다. 현재는 단순 이동 평균(SMA)을 계산하는 함수를 포함하고 있다.
*   `README.md`: 프로젝트에 대한 설명과 사용 방법을 기술한 문서이다.

## 실행 방법

이 프로젝트를 실행하려면 Node.js 환경이 필요하다.

1.  **의존성 설치**: 프로젝트에 필요한 라이브러리(`axios`)를 설치한다.
    ```bash
    npm install axios
    ```
2.  **API 키 설정 (선택 사항)**: `stock_api.js` 파일 내에 `ALPHA_VANTAGE_API_KEY`와 같은 API 키 변수가 있다면, 이를 자신의 API 키로 대체해야 한다. 이 예제에서는 더미 데이터를 사용하거나 API 키 없이 작동하도록 구성될 수 있다. 실제 API를 사용하려면 [Alpha Vantage](https://www.alphavantage.co/)와 같은 서비스에서 API 키를 발급받아야 한다.
3.  **애플리케이션 실행**: `main.js` 파일을 실행하여 주식 데이터 분석을 시작한다.
    ```bash
    node main.js
    ```

## 코드 구조 설명

각 파일은 다음과 같은 역할을 수행한다.

### `main.js`

`main.js`는 `stock_api.js`의 `fetchStockData` 함수를 호출하여 주식 데이터를 가져오고, `analyzer.js`의 `calculateSMA` 함수를 사용하여 이동 평균을 계산한다. 최종 결과를 콘솔에 출력한다.

### `stock_api.js`

`fetchStockData(symbol)` 함수는 주어진 `symbol`에 해당하는 주식의 일별 시계열 데이터를 API로부터 비동기적으로 가져온다. 실제 API 호출 대신 예제 데이터를 반환하도록 구현될 수 있다.

### `analyzer.js`

`calculateSMA(data, period)` 함수는 주어진 `data` 배열과 이동 평균 기간 `period`를 사용하여 단순 이동 평균을 계산한다. `data`는 가격 정보를 포함하는 객체 배열이어야 한다.

## 사용된 라이브러리 및 알고리즘

*   **`axios`**: HTTP 요청을 보내는 데 사용되는 Promise 기반 HTTP 클라이언트이다.
*   **단순 이동 평균 (Simple Moving Average, SMA)**: 주식 가격 데이터에서 일정 기간 동안의 평균값을 계산하여 추세를 파악하는 데 사용되는 기본적인 기술적 분석 지표이다.

이 프로젝트는 위에서 언급된 기능들을 구현한 소규모 예제이며, 더 복잡한 분석이나 데이터 시각화를 위해서는 추가적인 개발이 필요하다.
