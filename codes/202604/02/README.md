# 한국어 뉴스 기사 감성 분석 프로젝트

## 프로젝트 개요
이 프로젝트는 한국어 뉴스 기사 또는 문장에 대한 감성을 분석하는 간단한 파이썬 애플리케이션입니다. 사용자가 입력한 텍스트를 전처리하고, 미리 정의된 규칙 기반 모델을 통해 해당 텍스트의 긍정적, 중립적, 부정적 감성을 판단합니다. `Konlpy`와 같은 한국어 처리 라이브러리의 개념을 활용하여 텍스트 분석의 기초를 보여줍니다.

## 파일 구조
- `README.md`: 프로젝트 설명, 설치 및 실행 방법.
- `main.py`: 사용자 입력을 받아 감성 분석을 실행하고 결과를 출력하는 메인 스크립트입니다.
- `preprocessor.py`: 텍스트 전처리 (예: 형태소 분석, 불용어 제거) 기능을 담당하는 모듈입니다.
- `sentiment_model.py`: 감성 분석 로직을 포함하는 모듈입니다. 이 예시에서는 간단한 키워드 기반 감성 분류를 수행합니다.

## 설치 방법
이 프로젝트를 실행하기 위해 필요한 라이브러리들은 다음과 같습니다.

1.  **Python 3.x 설치**: 먼저 Python 3.x 버전이 시스템에 설치되어 있는지 확인하십시오.
2.  **가상 환경 설정 (권장)**:
    ```bash
    python -m venv venv
    .\venv\Scripts\activate   # Windows
    source venv/bin/activate # macOS/Linux
    ```
3.  **필요 라이브러리 설치**:
    `Konlpy`는 한국어 형태소 분석을 위해 필요하며, `JPype1`은 `Konlpy`의 Java 의존성을 위해 필요합니다.
    ```bash
    pip install konlpy
    pip install JPype1==1.4.1 # Konlpy 버전과 호환되는 JPype1 버전을 설치
    ```
    `Konlpy` 설치 시 Java Runtime Environment (JRE)가 필요할 수 있습니다. 시스템에 Java가 설치되어 있는지 확인하고, 설치되어 있지 않다면 OpenJDK 등을 설치해 주십시오.

## 실행 방법
모든 라이브러리 설치가 완료되었다면, `main.py` 스크립트를 실행하여 감성 분석을 시작할 수 있습니다.

```bash
python main.py
```

스크립트를 실행하면 텍스트 입력을 요청하는 프롬프트가 나타납니다. 분석하고 싶은 한국어 문장을 입력하고 Enter 키를 누르십시오.

## 코드 구조 설명

### `preprocessor.py`
- `Okt` 형태소 분석기를 사용하여 텍스트를 명사, 동사, 형용사 등으로 분리합니다.
- 간단한 불용어 리스트를 포함하여 분석에 불필요한 단어를 제거합니다.
- 전처리된 토큰 리스트를 반환합니다.

### `sentiment_model.py`
- 긍정 및 부정 키워드 리스트를 정의합니다.
- 전처리된 텍스트에서 긍정 및 부정 키워드의 출현 빈도를 계산합니다.
- 긍정 키워드가 더 많으면 '긍정적', 부정 키워드가 더 많으면 '부정적', 그렇지 않으면 '중립적'으로 분류합니다.

### `main.py`
- `preprocessor`와 `sentiment_model` 모듈의 함수를 가져옵니다.
- 사용자로부터 텍스트를 입력받습니다.
- 입력된 텍스트를 `preprocessor`로 전처리합니다.
- 전처리된 텍스트를 `sentiment_model`로 감성 분석합니다.
- 분석 결과를 사용자에게 출력합니다.
