# Pandas를 활용한 간단한 데이터 분석 프로젝트

## 프로젝트 개요
이 프로젝트는 Python의 강력한 데이터 분석 라이브러리인 Pandas를 사용하여 CSV 파일로부터 데이터를 로드하고, 전처리하며, 간단한 통계 분석을 수행하는 예시입니다. 데이터 분석의 기본적인 워크플로우를 이해하고 실습하는 데 중점을 둡니다.

## 파일 구조

- `main.py`: 프로젝트의 메인 실행 파일입니다. `data_processor.py`와 `analyzer.py` 모듈의 함수들을 호출하여 전체 데이터 분석 과정을 Orchestrate합니다.
- `data_processor.py`: 데이터 로딩 및 전처리 기능을 담당합니다. CSV 파일을 읽고, 필요한 데이터 형식 변환(예: 날짜 컬럼을 datetime 객체로)을 수행합니다.
- `analyzer.py`: 전처리된 데이터를 기반으로 다양한 통계 분석을 수행하는 기능을 포함합니다. 기술 통계 계산, 범주별 평균 분석, 상위 N개 값 찾기 등의 함수를 제공합니다.
- `sample_data.csv`: 분석을 위한 샘플 데이터가 포함된 CSV 파일입니다. 프로젝트 실행 시 이 파일을 로드하여 분석을 진행합니다.

## 실행 방법

1.  **필수 라이브러리 설치**:
    이 프로젝트는 `pandas` 라이브러리에 의존합니다. 다음 명령어를 사용하여 설치할 수 있습니다.
    ```bash
    pip install pandas
    ```

2.  **프로젝트 실행**:
    프로젝트의 루트 디렉터리(이 `README.md` 파일이 있는 곳)에서 다음 명령어를 실행합니다.
    ```bash
    python main.py
    ```

## 분석 결과 예시

`main.py`를 실행하면 콘솔에 다음과 같은 분석 결과가 출력됩니다.

-   원본 데이터 미리보기
-   전처리된 데이터 정보 (컬럼 타입, 결측치 여부 등)
-   특정 컬럼(예: 'Value')의 기술 통계 (평균, 표준편차, 최소/최대값 등)
-   범주별(예: 'Category') 특정 값(예: 'Value')의 평균
-   특정 값(예: 'Value')을 기준으로 상위 N개 데이터

## 프로젝트 확장 아이디어

-   **데이터 시각화**: `matplotlib` 또는 `seaborn` 라이브러리를 사용하여 분석 결과를 그래프로 시각화할 수 있습니다.
-   **더 복잡한 전처리**: 결측치 처리, 이상치 제거, 데이터 정규화 등 다양한 전처리 기법을 추가할 수 있습니다.
-   **고급 분석 기법**: 회귀 분석, 클러스터링, 시계열 분석 등 더 고급 통계 및 머신러닝 기법을 적용할 수 있습니다.
-   **대시보드 구축**: `Dash`나 `Streamlit` 같은 라이브러리를 활용하여 인터랙티브한 분석 대시보드를 만들 수 있습니다.
