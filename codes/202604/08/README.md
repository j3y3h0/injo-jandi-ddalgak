# C# 데이터 분석 예제 프로젝트

이 프로젝트는 C#을 사용하여 CSV 파일로부터 데이터를 읽고 기본적인 통계 분석(평균, 최소값, 최대값)을 수행하는 소규모 예제 애플리케이션입니다. 산업에서 흔히 사용되는 데이터 처리 및 분석의 기초를 보여주는 실용적인 코드입니다.

## 코드 구조

*   **`Program.cs`**: 애플리케이션의 진입점입니다. `ExampleData.csv` 파일을 읽어 `DataProcessor` 클래스를 통해 분석을 수행하고, 그 결과를 콘솔에 출력합니다.
*   **`DataProcessor.cs`**: 데이터 처리 로직을 캡슐화한 클래스입니다. CSV 파일의 경로를 받아 데이터를 파싱하고, 숫자 데이터에 대한 통계 정보(평균, 최소, 최대)를 계산하는 메서드를 제공합니다.
*   **`ExampleData.csv`**: 이 프로젝트에서 사용될 샘플 데이터 파일입니다. 쉼표로 구분된 값(CSV) 형식으로 구성되어 있으며, `값`이라는 헤더 아래에 분석할 숫자 데이터가 포함되어 있습니다.
*   **`DataAnalysis.csproj`**: C# 프로젝트 파일로, 프로젝트의 빌드 설정 및 종속성을 정의합니다.

## 실행 방법

이 프로젝트를 실행하려면 .NET SDK가 설치되어 있어야 합니다.

1.  **프로젝트 디렉터리로 이동:**
    ```bash
    cd C:\server\gpt-code-diary\codes\202604\08
    ```

2.  **애플리케이션 실행:**
    ```bash
    dotnet run
    ```

위 명령어를 실행하면 `Program.cs`가 `DataProcessor`를 사용하여 `ExampleData.csv`를 분석하고, 분석된 통계 결과를 콘솔에 출력합니다.

## 주요 사용 라이브러리/알고리즘

*   **`System.IO`**: 파일 입출력을 처리합니다. CSV 파일을 읽는 데 사용됩니다.
*   **`System.Linq`**: 컬렉션에 대한 쿼리 작업을 위한 강력한 기능을 제공합니다. 이 프로젝트에서는 `Average()`, `Min()`, `Max()` 등의 메서드를 사용하여 통계 값을 계산하는 데 활용됩니다.
*   **데이터 파싱**: CSV 파일에서 문자열 형태의 데이터를 숫자로 변환하는 기본적인 파싱 로직이 포함되어 있습니다.
