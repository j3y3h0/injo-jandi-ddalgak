# 간단한 판매 데이터 분석기 (Sales Analyzer)

이 프로젝트는 JSON 파일로부터 판매 데이터를 읽어와, 제품별 총 판매액을 분석하고 결과를 콘솔에 출력하는 간단한 C# 애플리케이션입니다. 데이터 분석 및 처리의 기본적인 패턴을 보여주며, `.NET` 환경에서 `System.Text.Json`을 이용한 JSON 처리와 `LINQ`를 활용한 데이터 집계 방법을 실용적인 예제로 제시합니다.

## 주요 기능

*   JSON 형식의 판매 데이터 파일을 읽어옵니다.
*   각 제품의 총 판매액을 계산합니다.
*   분석된 결과를 제품별 총 판매액이 높은 순으로 정렬하여 콘솔에 출력합니다.
*   파일이 없거나 데이터 처리 중 오류가 발생할 경우 적절한 오류 메시지를 제공합니다.

## 프로젝트 구조

```
.
├── SalesAnalyzer.csproj     # C# 프로젝트 파일
├── Program.cs             # 애플리케이션의 진입점 및 실행 로직
├── Models.cs              # 판매 기록 및 분석 결과를 위한 데이터 모델 정의
├── DataProcessor.cs       # 데이터 로드 및 분석 로직 구현
├── SampleData.json        # 분석에 사용될 샘플 JSON 판매 데이터
└── README.md              # 프로젝트 설명 및 사용 방법 (현재 파일)
```

### 각 파일의 역할

*   **`SalesAnalyzer.csproj`**: 이 파일은 프로젝트를 빌드하는 데 필요한 설정과 종속성을 정의하는 C# 프로젝트 파일입니다. .NET 8.0을 대상으로 합니다.
*   **`Program.cs`**: 애플리케이션의 시작 지점입니다. `DataProcessor` 클래스를 사용하여 `SampleData.json`에서 데이터를 로드하고 분석한 후, 결과를 콘솔에 출력하는 역할을 합니다.
*   **`Models.cs`**: `SaleRecord` (개별 판매 기록)와 `ProductSales` (제품별 총 판매액) 두 가지 데이터 모델을 정의합니다. `System.Text.Json.Serialization`의 `JsonPropertyName` 속성을 사용하여 JSON 속성 이름과 C# 속성 이름을 매핑합니다.
*   **`DataProcessor.cs`**: 이 클래스는 판매 데이터 로드 및 분석의 핵심 로직을 포함합니다. `LoadSalesData` 메서드는 JSON 파일을 비동기적으로 읽어 `SaleRecord` 객체 리스트로 역직렬화하고, `AnalyzeSalesData` 메서드는 이 리스트를 받아 `LINQ`를 사용하여 제품별 총 판매액을 집계합니다.
*   **`SampleData.json`**: 애플리케이션이 로드하여 분석할 샘플 판매 데이터가 JSON 형식으로 저장되어 있습니다.

이 프로젝트는 .NET 8.0 SDK가 설치된 환경에서 실행할 수 있습니다.

1.  **프로젝트 디렉토리로 이동**:
    터미널 또는 명령 프롬프트에서 프로젝트 파일(`SalesAnalyzer.csproj`)이 있는 디렉토리로 이동합니다.

    ```bash
    cd C:\server\gpt-code-diary\codes\202604\10
    ```

2.  **애플리케이션 실행**:
    다음 명령어를 사용하여 애플리케이션을 실행합니다.

    ```bash
    dotnet run
    ```

    `dotnet run` 명령은 프로젝트를 자동으로 빌드하고 실행합니다.
    실행이 완료되면 콘솔에 제품별 총 판매액 분석 결과가 출력됩니다.

## 사용된 기술

*   **C# 8.0+**: .NET 애플리케이션 개발 언어.
*   **.NET 8.0 SDK**: 애플리케이션 빌드 및 실행 환경.
*   **System.Text.Json**: 고성능 JSON 직렬화/역직렬화 라이브러리.
*   **LINQ (Language Integrated Query)**: 데이터 쿼리 및 집계를 위한 강력한 기능.
