# 간단한 데이터 처리 및 분석 유틸리티 (Simple Data Processing and Analysis Utility)

## 프로젝트 설명

이 프로젝트는 C# 콘솔 애플리케이션으로, LINQ(Language Integrated Query)를 활용하여 숫자 데이터 목록에 대한 기본적인 처리 및 분석 작업을 수행하는 방법을 보여줍니다. 중복 제거, 정렬, 필터링, 합계 및 평균 계산과 같은 일반적인 데이터 조작 기능을 제공합니다. 이 유틸리티는 산업에서 널리 사용되는 데이터 처리 패턴을 시뮬레이션하며, C#에서 데이터를 효율적으로 다루는 실용적인 예시를 제시합니다.

## 코드 구조

프로젝트는 다음 두 가지 주요 C# 파일로 구성되어 있습니다.

*   `SimpleDataProcessor.csproj`: C# 프로젝트 파일로, 프로젝트의 빌드 설정 및 종속성을 정의합니다. .NET 8.0을 대상으로 합니다.
*   `Program.cs`: 애플리케이션의 진입점입니다. `List<int>` 형태의 샘플 데이터를 정의하고, `DataProcessor` 클래스의 메서드를 호출하여 다양한 데이터 처리 작업을 수행한 후 그 결과를 콘솔에 출력합니다.
*   `DataProcessor.cs`: `static` 클래스로, 데이터 처리 로직을 캡슐화합니다. LINQ 확장 메서드(`Distinct`, `OrderBy`, `Where`, `Sum`, `Average`, `Any`)를 사용하여 데이터를 조작하는 정적 메서드들을 포함합니다.

## 주요 기능

*   **중복 제거 및 정렬:** `List<int>`에서 중복된 요소를 제거하고 오름차순으로 정렬합니다.
*   **데이터 필터링:** 특정 기준(예: 특정 값보다 큰 숫자)에 따라 데이터를 필터링합니다.
*   **데이터 집계:** 숫자 목록의 합계와 평균을 계산합니다.

이 프로젝트는 C#과 LINQ를 이용한 실용적인 데이터 처리 기법을 학습하거나 기본적인 데이터 분석 기능을 필요로 하는 작은 유틸리티를 개발할 때 유용한 참고 자료가 될 수 있습니다.
