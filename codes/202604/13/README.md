# 센서 데이터 분석기 (Sensor Data Analyzer)

이 프로젝트는 시뮬레이션된 센서 데이터를 읽고, 기본적인 통계 분석(최소값, 최대값, 평균)을 수행하며, 잠재적인 이상치를 탐지하는 소규모 C# 콘솔 애플리케이션입니다. 산업 환경에서 센서 데이터를 모니터링하고 분석하는 시나리오를 가정하여 개발되었습니다.

## 프로젝트 구조

*   `Program.cs`: 애플리케이션의 진입점입니다. 파일로부터 데이터를 로드하고, `DataAnalyzer`를 사용하여 분석하며, 결과를 콘솔에 출력하는 역할을 합니다.
*   `SensorData.cs`: 단일 센서 측정값의 데이터 모델을 정의합니다. `DateTime` 타입의 타임스탬프와 `double` 타입의 값으로 구성됩니다.
*   `DataAnalyzer.cs`: 센서 데이터 목록에 대한 통계 계산(평균, 최소, 최대) 및 간단한 이상치 탐지 로직을 포함합니다.
*   `FileHandler.cs`: CSV 파일에서 센서 데이터를 읽거나 쓰는 기능을 제공합니다. 데이터 영속성을 관리합니다.

## 실행 방법

1.  **프로젝트 빌드**:
    명령 프롬프트 또는 터미널에서 다음 명령을 실행하여 프로젝트를 빌드합니다.
    ```bash
    dotnet build
    ```

2.  **샘플 데이터 파일 준비**:
    프로젝트의 실행 디렉터리(일반적으로 `bin/Debug/netX.Y/` 폴더)에 `sensor_data.csv` 파일을 생성합니다. 이 파일은 다음과 같은 형식이어야 합니다:
    ```
    2026-04-13 10:00:00,25.1
    2026-04-13 10:01:00,25.3
    2026-04-13 10:02:00,35.0 ; 이상치 예시
    2026-04-13 10:03:00,25.2
    ```
    여기서 각 줄은 `타임스탬프,값` 형식입니다.

3.  **애플리케이션 실행**:
    프로젝트가 성공적으로 빌드되면, 다음 명령을 사용하여 애플리케이션을 실행할 수 있습니다.
    ```bash
    dotnet run
    ```
    애플리케이션은 `sensor_data.csv` 파일을 읽어 분석을 수행하고, 결과를 콘솔에 출력합니다.

## 사용된 기술 및 알고리즘

*   **C# .NET**: 애플리케이션 개발에 사용된 주 언어 및 프레임워크입니다.
*   **파일 I/O**: `System.IO` 네임스페이스를 활용하여 CSV 파일을 읽고 씁니다.
*   **통계 분석**: `System.Linq`를 사용하여 데이터 집합에 대한 평균, 최소값, 최대값 등을 효율적으로 계산합니다.
*   **간단한 이상치 탐지**: 평균에서 표준 편차의 특정 배수(예: 2배) 이상 벗어나는 데이터를 이상치로 간주하는 간단한 휴리스틱을 사용합니다. (현재 구현에서는 간단한 Min/Max/Avg 기반의 이상치 탐지를 제공하며, 향후 표준편차 기반으로 확장 가능합니다.)

이 프로젝트는 실제 산업용 애플리케이션의 기반이 될 수 있으며, 더 복잡한 통계 모델, 실시간 데이터 처리, 시각화 기능 등으로 확장될 수 있습니다.
