# 데이터 분석 및 시각화 예제 프로젝트

## 프로젝트 개요
이 프로젝트는 간단한 자바스크립트 기반의 데이터 분석 예제입니다. CSV 형식의 데이터를 읽어와 특정 컬럼에 대한 합계와 평균을 계산하는 기능을 포함하고 있습니다. 실제 데이터 분석 및 시각화 워크플로우의 기초적인 단계를 보여주기 위해 만들어졌습니다.

## 파일 구성
*   `README.md`: 현재 문서. 프로젝트에 대한 설명과 실행 방법을 담고 있습니다.
*   `main.js`: 애플리케이션의 주 진입점입니다. `dataProcessor.js` 모듈을 사용하여 데이터를 처리하고 결과를 콘솔에 출력합니다.
*   `dataProcessor.js`: 데이터 처리 로직을 포함하는 유틸리티 모듈입니다. CSV 문자열을 파싱하고, 지정된 컬럼의 합계와 평균을 계산하는 함수들을 제공합니다.
*   `sample_data.js`: 분석에 사용될 샘플 데이터를 문자열 형태로 담고 있습니다.

## 코드 구조 설명
### `main.js`
`dataProcessor.js`에서 제공하는 `parseCsv`와 `calculateColumnStatistics` 함수를 임포트하여 사용합니다. `sample_data.js`의 데이터를 불러와 처리한 후, 결과를 사용자에게 친숙한 형태로 보여줍니다.

### `dataProcessor.js`
*   `parseCsv(csvString)`: CSV 형식의 문자열을 받아 각 행을 배열로, 각 행의 요소를 배열로 변환하는 함수입니다. 첫 번째 행은 헤더로 간주됩니다.
*   `calculateColumnStatistics(data, columnName)`: 파싱된 데이터와 통계를 계산할 컬럼 이름을 인자로 받아, 해당 컬럼의 합계(sum)와 평균(average)을 계산하여 반환합니다.

### `sample_data.js`
`exports.sampleCsvData` 변수에 임의의 CSV 데이터를 문자열 형태로 저장하고 있습니다.

## 실행 방법
이 프로젝트는 Node.js 환경에서 실행됩니다.
1.  프로젝트 디렉터리로 이동합니다.
2.  다음 명령어를 터미널에서 실행하여 `main.js` 파일을 실행합니다:
    ```bash
    node main.js
    ```
3.  콘솔에 분석 결과가 출력되는 것을 확인할 수 있습니다.

## 활용 가능한 산업 라이브러리 및 확장
현재 프로젝트는 핵심 로직만을 포함하고 있습니다. 실제 산업 환경에서는 다음과 같은 라이브러리들을 활용하여 기능을 확장할 수 있습니다:
*   **데이터 파싱**: `csv-parser`, `papaparse` 등
*   **데이터 분석**: `lodash`, `d3-array` (D3.js의 배열 유틸리티) 등
*   **데이터 시각화**: `D3.js`, `Chart.js`, `Plotly.js` 등
*   **통계 및 머신러닝**: `TensorFlow.js`, `brain.js` 등

이 프로젝트는 이러한 라이브러리들을 활용하기 위한 기본적인 데이터 처리 파이프라인의 이해를 돕기 위한 출발점이 됩니다.
