# 텍스트 감성 분석 프로젝트

## 프로젝트 개요
이 프로젝트는 입력된 한국어 텍스트의 감성을 긍정, 부정, 중립으로 분류하는 간단한 감성 분석 프로그램입니다. 텍스트 처리 및 감성 분석 로직을 모듈화하여 구성하였습니다.

## 파일 구조
- `main.py`: 프로그램의 주요 실행 파일입니다. 사용자로부터 텍스트를 입력받아 감성 분석을 수행하고 결과를 출력합니다.
- `sentiment_analyzer.py`: 감성 분석을 담당하는 핵심 로직을 포함합니다. 여기서는 간단한 키워드 기반 규칙을 사용하여 감성을 판단합니다.
- `utils.py`: 텍스트 전처리(예: 특수문자 제거)와 같은 유틸리티 함수들을 포함합니다.
- `requirements.txt`: 이 프로젝트를 실행하는 데 필요한 Python 라이브러리 목록입니다.

## 실행 방법
1. **환경 설정**: Python이 설치되어 있는지 확인하십시오.
2. **필요 라이브러리 설치**: 현재 프로젝트는 기본 Python 라이브러리만 사용하므로 별도의 설치가 필요 없습니다. 만약 `requirements.txt`에 라이브러리가 추가된다면 다음 명령어를 사용하여 설치합니다.
   ```bash
   pip install -r requirements.txt
   ```
3. **프로그램 실행**: `main.py` 파일을 실행하여 프로그램을 시작합니다.
   ```bash
   python main.py
   ```
4. **텍스트 입력**: 프로그램이 실행되면 감성을 분석하고자 하는 텍스트를 입력하라는 메시지가 나타납니다. 텍스트를 입력하고 Enter 키를 누르면 감성 분석 결과가 출력됩니다. 종료하려면 'exit'을 입력하십시오.

## 코드 설명
- `sentiment_analyzer.py` 파일 내 `analyze_sentiment` 함수는 미리 정의된 긍정 및 부정 키워드 목록을 기반으로 텍스트에 포함된 키워드를 찾아 감성을 결정합니다. 키워드 점수 합산 방식을 사용하며, 향후 더 정교한 머신러닝 모델(예: BERT, KoELECTRA 등)로 확장할 수 있습니다.
- `utils.py` 파일 내 `preprocess_text` 함수는 입력 텍스트에서 불필요한 문자(예: 구두점, 숫자)를 제거하여 분석 정확도를 높이는 역할을 합니다.
