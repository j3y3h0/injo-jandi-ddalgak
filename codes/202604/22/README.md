# 간단한 웹 스크래퍼 프로젝트

이 프로젝트는 `axios`와 `cheerio` 라이브러리를 활용하여 웹 페이지에서 특정 정보를 추출하는 간단한 웹 스크래퍼 예제입니다. 자바스크립트로 작성되었으며, 모듈 방식으로 구성되어 있습니다.

## 프로젝트 구조

*   `package.json`: 프로젝트의 메타데이터와 의존성(dependency) 정보를 정의하는 파일입니다. `axios`와 `cheerio` 라이브러리가 포함되어 있습니다.
*   `main.js`: 이 스크래퍼의 메인 실행 파일입니다. `scraper.js`의 함수들을 호출하여 웹 페이지에서 데이터를 가져오고, `utils.js`의 함수를 사용하여 추출된 데이터를 파일로 저장합니다.
*   `scraper.js`: 실제 웹 스크래핑 로직을 담고 있는 모듈입니다. `fetchAndParse` 함수를 통해 지정된 URL의 HTML을 가져오고 `cheerio`를 사용하여 파싱하며, `extractProductInfo` 함수를 통해 파싱된 HTML에서 상품 정보를 추출합니다.
*   `utils.js`: 유틸리티 함수들을 모아둔 모듈입니다. 현재는 `saveDataToFile` 함수를 포함하고 있으며, 이는 추출된 데이터를 JSON 파일로 저장하는 역할을 합니다.
*   `products.json`: (스크래퍼 실행 후 생성됨) 스크래핑을 통해 추출된 상품 정보가 저장되는 파일입니다.

## 사용된 기술

*   **Node.js**: 자바스크립트 런타임 환경입니다.
*   **axios**: HTTP 요청을 보내는 데 사용되는 Promise 기반의 HTTP 클라이언트 라이브러리입니다.
*   **cheerio**: 서버 측에서 jQuery와 유사한 구문으로 HTML을 파싱하고 조작하는 데 사용되는 라이브러리입니다.

## 실행 방법

1.  **프로젝트 클론 및 이동**:
    ```bash
    # 이 프로젝트가 위치한 디렉터리로 이동합니다.
    cd C:\server\gpt-code-diary\codes\202604\22
    ```

2.  **의존성 설치**:
    프로젝트에 필요한 Node.js 패키지들을 설치해야 합니다. `package.json`에 정의된 `axios`와 `cheerio`를 설치합니다.
    ```bash
    npm install
    ```

3.  **스크래퍼 실행**:
    설치가 완료되면 `main.js` 파일을 실행하여 웹 스크래핑을 시작할 수 있습니다.
    ```bash
    npm start
    ```
    또는
    ```bash
    node main.js
    ```

4.  **결과 확인**:
    스크래퍼 실행이 완료되면, 현재 디렉터리에 `products.json` 파일이 생성됩니다. 이 파일에는 스크래핑을 통해 추출된 상품 정보가 JSON 형식으로 저장되어 있습니다.

## 참고 사항

*   `scraper.js` 파일의 `extractProductInfo` 함수와 `main.js` 파일의 `TARGET_URL`은 스크래핑하려는 실제 웹 페이지의 구조와 URL에 맞춰 수정해야 합니다. 현재 `main.js`에는 더미 HTML이 포함되어 있어 실제 웹 페이지에 접속하지 않고도 테스트할 수 있습니다.
*   웹 스크래핑 시에는 해당 웹사이트의 `robots.txt` 파일을 확인하고, 서비스 약관을 준수해야 합니다. 과도한 요청은 서버에 부담을 줄 수 있으므로 주의해야 합니다.
