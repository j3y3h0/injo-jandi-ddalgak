# 비동기 데이터 처리 및 시각화 프로젝트

이 프로젝트는 JavaScript를 사용하여 외부 데이터를 비동기적으로 가져오고, 가공한 후, Chart.js 라이브러리를 활용하여 시각화하는 간단한 웹 애플리케이션입니다. 실제 산업 환경에서 데이터 처리 및 대시보드 구축 시 사용될 수 있는 기본적인 흐름을 보여줍니다.

## 프로젝트 목표

*   비동기 JavaScript (async/await)를 사용한 데이터 가져오기 구현.
*   가져온 원시 데이터의 가공 및 집계.
*   Chart.js 라이브러리를 이용한 데이터 시각화.
*   모듈화를 통한 코드의 구조화.

## 코드 구조

프로젝트는 다음 파일들로 구성되어 있습니다.

*   `index.html`: 웹 페이지의 기본 구조를 정의하고, Chart.js 라이브러리와 애플리케이션의 메인 JavaScript 모듈을 로드합니다. 데이터 시각화를 위한 캔버스 요소와 데이터 로드를 트리거하는 버튼을 포함하고 있습니다.
*   `main.js`: 애플리케이션의 진입점입니다. `dataFetcher.js`, `dataProcessor.js`, `visualization.js` 모듈을 임포트하여 데이터 가져오기, 가공, 시각화의 전 과정을 조정합니다. "데이터 불러오기" 버튼 클릭 이벤트에 반응하여 이 모든 작업을 비동기적으로 실행합니다.
*   `dataFetcher.js`: 가상의 외부 API로부터 데이터를 비동기적으로 가져오는 역할을 하는 모듈입니다. `fetchData` 함수를 제공하며, 현재는 2초의 지연 시간을 가진 목(mock) 데이터를 반환하도록 구현되어 있습니다. 실제 API 연동 시 `fetch` API를 사용하여 실제 엔드포인트와 통신하도록 확장될 수 있습니다.
*   `dataProcessor.js`: `dataFetcher.js`에서 가져온 원시 데이터를 시각화에 적합한 형태로 가공하는 모듈입니다. `processData` 함수를 제공하며, 현재는 데이터의 'category'별로 'value'를 집계하여 차트 데이터 형식으로 변환합니다.
*   `visualization.js`: `dataProcessor.js`에 의해 가공된 데이터를 받아 Chart.js 라이브러리를 사용하여 시각화하는 모듈입니다. `renderChart` 함수를 제공하며, 주어진 캔버스 요소에 막대 차트를 그립니다. 기존 차트 인스턴스가 있다면 파괴하고 새로 그리는 방식으로 업데이트를 처리합니다.

## 실행 방법

1.  **파일 열기**: `index.html` 파일을 웹 브라우저 (Chrome, Firefox, Edge 등)로 엽니다.
2.  **데이터 불러오기**: 웹 페이지에 나타난 "데이터 불러오기" 버튼을 클릭합니다.
3.  **시각화 확인**: 버튼을 클릭하면 `dataFetcher.js`에서 2초 후에 가상의 데이터를 불러오고, `dataProcessor.js`에서 데이터를 가공한 후, `visualization.js`를 통해 Chart.js로 막대 차트가 화면에 나타나는 것을 확인할 수 있습니다.
4.  **개발자 도구 활용**: 브라우저의 개발자 도구(F12) 콘솔 탭을 열면 데이터 가져오기, 가공, 시각화 과정의 로그를 확인할 수 있습니다.

이 프로젝트는 모듈식 JavaScript 개발의 기초와 비동기 데이터 처리, 그리고 외부 라이브러리(Chart.js)를 활용한 웹 기반 시각화의 예시를 제공합니다.
