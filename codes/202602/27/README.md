# 간단한 데이터 분석 및 분류 도구

이 프로젝트는 주어진 CSV 형식의 데이터를 파싱하고, 기본적인 통계 분석을 수행하며, 간단한 규칙 기반으로 데이터를 분류하는 작은 JavaScript 애플리케이션입니다.

## 코드 구조

*   `main.js`: 애플리케이션의 주 진입점이다. 데이터를 로드하고, `dataParser`, `dataAnalyzer`, `dataClassifier` 모듈을 사용하여 데이터를 처리한 후 결과를 출력한다.
*   `dataParser.js`: CSV 형식의 문자열 데이터를 JavaScript 객체 배열로 변환하는 기능을 제공한다.
*   `dataAnalyzer.js`: 숫자 데이터 배열을 받아 평균, 중앙값, 최댓값, 최솟값 등 기본적인 통계 지표를 계산하는 함수들을 포함한다.
*   `dataClassifier.js`: 특정 기준(예: 임계값)에 따라 데이터를 '높음', '중간', '낮음' 등으로 분류하는 간단한 규칙 기반 분류기를 구현한다.

## 실행 방법

1.  프로젝트 디렉터리로 이동한다.
2.  `main.js` 파일을 Node.js 런타임으로 실행한다.

    ```bash
    node main.js
    ```

## 사용된 기술 및 라이브러리 (개념적)

본 프로젝트는 순수 JavaScript만을 사용하여 구현되었다. 실제 산업 환경에서는 더 복잡하고 효율적인 데이터 처리 및 분석을 위해 다음과 같은 라이브러리들이 활용될 수 있다:

*   **데이터 파싱:** `csv-parser`, `papaparse` 등
*   **데이터 조작 및 통계:** `lodash`, ` estadísticas.js` 등
*   **머신러닝:** `TensorFlow.js`, `Brain.js` (더 복잡한 분류 모델의 경우)

이 프로젝트는 위 라이브러리들의 핵심 개념을 JavaScript 내장 기능으로 간략하게 구현하여 보여주는 예시이다.
