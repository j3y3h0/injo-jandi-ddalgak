# 자바 합병 정렬 데모 프로젝트이다.

이 프로젝트는 자바(Java) 언어로 구현된 합병 정렬(Merge Sort) 알고리즘을 시연하는 소규모 애플리케이션이다. 무작위로 생성된 정수 배열을 합병 정렬을 사용하여 정렬하고, 정렬에 소요된 시간을 측정하여 출력한다.

## 파일 구조이다.

*   `MergeSort.java`: 합병 정렬 알고리즘의 핵심 로직이 구현된 파일이다. 재귀적인 분할(Divide)과 정복(Conquer) 방식을 사용하여 배열을 정렬한다.
*   `Main.java`: 프로그램의 진입점(Entry Point)이다. 다음 작업을 수행한다.
    *   지정된 크기(기본값 100,000)의 정수 배열을 생성하고 무작위 숫자로 채운다.
    *   `MergeSort` 클래스를 사용하여 배열을 정렬한다.
    *   정렬 전후 배열의 일부를 출력하여 정렬 결과를 보여준다.
    *   정렬 작업에 소요된 시간을 밀리초(ms) 단위로 측정하여 출력한다.
    *   배열이 올바르게 정렬되었는지 확인하는 간단한 검증 로직을 포함한다.
*   `README.md`: 이 문서이다. 프로젝트에 대한 설명과 사용 방법을 제공한다.

## 프로젝트 실행 방법이다.

이 프로젝트는 표준 자바 개발 키트(JDK)를 사용하여 컴파일하고 실행할 수 있다.

1.  **자바 개발 키트(JDK) 설치이다.**
    아직 JDK가 설치되어 있지 않다면, [Oracle JDK](https://www.oracle.com/java/technologies/downloads/) 또는 [OpenJDK](https://openjdk.java.net/install/index.html) 웹사이트에서 최신 버전을 다운로드하여 설치해야 한다.

2.  **소스 코드 컴파일이다.**
    명령 프롬프트(Command Prompt) 또는 터미널을 열고, 프로젝트 파일들이 위치한 디렉터리(`C:\server\gpt-code-diary\codes\202602\12`)로 이동한다. 그런 다음, 다음 명령어를 사용하여 자바 소스 파일을 컴파일한다.

    ```bash
    javac Main.java MergeSort.java
    ```
    이 명령어는 `Main.java`와 `MergeSort.java` 파일을 컴파일하여 각각 `Main.class`와 `MergeSort.class`라는 바이트코드 파일을 생성한다.

3.  **프로그램 실행이다.**
    컴파일이 완료되면, 다음 명령어를 사용하여 프로그램을 실행할 수 있다.

    ```bash
    java Main
    ```
    프로그램은 무작위 숫자로 채워진 배열을 정렬하고, 그 결과를 콘솔에 출력할 것이다. 출력에는 정렬 전후 배열의 일부와 정렬에 소요된 시간이 포함된다.

## 코드 주석이다.

모든 자바 소스 코드 파일(`MergeSort.java`, `Main.java`)에는 코드의 목적과 작동 방식을 설명하는 한국어 주석이 상세하게 포함되어 있다. 이를 통해 코드의 이해도를 높일 수 있다.
