# CsvProcessor 프로젝트

이 프로젝트는 C#으로 구현된 간단한 CSV 파일 처리기입니다. CSV 파일을 읽고, 데이터를 처리하며, 특정 조건에 따라 필터링하는 기능을 제공합니다.

## 프로젝트 구조

*   **`Program.cs`**:
    *   이 프로젝트의 진입점입니다.
    *   `CsvReader`와 `DataProcessor` 클래스를 사용하여 CSV 파일을 읽고 데이터를 처리하는 과정을 보여줍니다.
    *   `sample_data.csv` 파일이 존재하지 않을 경우 자동으로 생성합니다.
    *   처리된 데이터의 평균을 계산하고, 특정 카테고리로 필터링된 데이터를 콘솔에 출력합니다.

*   **`CsvReader.cs`**:
    *   CSV 파일을 읽고 파싱하는 기능을 담당하는 유틸리티 클래스입니다.
    *   파일 경로를 입력받아 CSV 내용을 `List<Dictionary<string, string>>` 형태로 반환합니다.
    *   첫 번째 줄을 헤더로 인식하며, 각 레코드를 헤더 이름을 키로 하는 사전(Dictionary)으로 변환합니다.

*   **`DataProcessor.cs`**:
    *   데이터에 대한 다양한 처리 기능을 제공하는 유틸리티 클래스입니다.
    *   `CalculateAverage` 메서드는 지정된 컬럼의 숫자 값들에 대한 평균을 계산합니다.
    *   `FilterData` 메서드는 지정된 컬럼의 값이 특정 기준 값과 일치하는 레코드들을 필터링합니다.

## 실행 방법

이 프로젝트는 .NET Core 환경에서 실행됩니다. 프로젝트를 실행하려면 다음 단계를 따르십시오.

1.  **`.csproj` 파일 생성**: 프로젝트 폴더에서 명령 프롬프트나 터미널을 열고 다음 명령어를 실행하여 새로운 C# 프로젝트를 초기화합니다.
    ```bash
    dotnet new console
    ```
    이 명령어는 `CsvProcessor.csproj` 파일과 기본 `Program.cs` 파일을 생성합니다. (기존 `Program.cs` 파일은 이 프로젝트에서 제공하는 파일로 덮어쓰거나 내용을 붙여넣어야 합니다.)

2.  **소스 파일 복사**: 제공된 `Program.cs`, `CsvReader.cs`, `DataProcessor.cs` 파일을 `.csproj` 파일이 있는 동일한 디렉터리에 복사합니다.

3.  **프로젝트 실행**: 프로젝트 폴더에서 다음 명령어를 실행하여 애플리케이션을 빌드하고 실행합니다.
    ```bash
    dotnet run
    ```

성공적으로 실행되면, `sample_data.csv` 파일이 자동으로 생성되고, 해당 파일의 내용이 읽혀 처리된 결과가 콘솔에 출력될 것입니다.

## 사용된 산업 라이브러리 및 알고리즘

*   **.NET Standard 라이브러리**: `System.IO` (파일 입출력), `System.Collections.Generic` (컬렉션), `System.Linq` (데이터 질의) 등 C#의 기본 및 표준 라이브러리를 활용합니다. 이는 C# 개발에서 가장 보편적으로 사용되는 핵심 기능들입니다.
*   **CSV 파싱**: CSV 파일의 헤더와 데이터를 분리하여 구조화된 데이터(사전 리스트)로 변환하는 일반적인 데이터 파싱 기법을 사용합니다.
*   **데이터 집계 (평균)**: 숫자 데이터 컬럼에 대한 평균을 계산하는 기본적인 통계 알고리즘을 사용합니다.
*   **데이터 필터링**: 특정 조건(컬럼 값 일치)에 따라 데이터를 선택하는 필터링 알고리즘을 사용하며, 이는 데이터베이스 질의나 일반적인 데이터 분석에서 널리 활용되는 기법입니다.
