# Java 기반 URL 단축 서비스 프로젝트

이 프로젝트는 긴 URL을 짧은 코드로 변환하고, 그 짧은 코드를 다시 원래의 긴 URL로 확장하는 간단한 URL 단축 서비스의 예시이다. 인메모리(In-Memory) 방식으로 데이터를 저장하며, 고유 ID를 Base62 문자열로 인코딩하여 짧은 URL을 생성한다.

## 프로젝트 구조

*   `Main.java`: 애플리케이션의 진입점이다. `UrlShortener` 클래스를 사용하여 여러 URL을 단축하고 확장하는 과정을 시연한다.
*   `UrlShortener.java`: URL 단축 서비스의 핵심 로직을 포함하는 클래스이다. 긴 URL과 짧은 코드 간의 매핑을 관리하며, 고유 ID를 생성하고 이를 Base62 문자열로 변환하는 기능을 제공한다.

## 사용 언어 및 기술

*   **언어**: Java
*   **데이터 저장**: `java.util.HashMap` (인메모리 방식)
*   **단축 알고리즘**: 고유한 숫자 ID를 Base62 (숫자, 알파벳 소문자, 알파벳 대문자) 문자열로 인코딩하는 방식이다.

## 실행 방법

이 프로젝트는 별도의 빌드 도구(예: Maven, Gradle) 없이 표준 Java 컴파일러와 JVM만으로 실행할 수 있다.

1.  **컴파일**: 터미널 또는 명령 프롬프트에서 프로젝트 디렉터리로 이동하여 다음 명령어를 실행하여 Java 소스 파일을 컴파일한다.
    ```bash
    javac Main.java UrlShortener.java
    ```
    *   `javac`는 Java 컴파일러이다. 이 명령어는 `.java` 파일을 `.class` 파일로 변환한다.

2.  **실행**: 컴파일이 완료되면 다음 명령어를 사용하여 애플리케이션을 실행한다.
    ```bash
    java Main
    ```
    *   `java`는 Java 가상 머신(JVM)을 실행하는 명령어이다. `Main` 클래스의 `main` 메서드가 호출되어 프로그램이 시작된다.

위 단계를 따르면 콘솔에 URL 단축 및 확장 결과가 출력될 것이다.
