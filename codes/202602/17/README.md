# 한국어 텍스트 감성 분석 프로젝트

## 프로젝트 개요

이 프로젝트는 한국어 텍스트에 대한 감성 분석 모델을 구축하고 활용하는 예시를 제공합니다. 자연어 처리(NLP) 기술을 사용하여 텍스트의 긍정적 또는 부정적 감성을 식별하는 것을 목표로 합니다. 주요 목표는 다음과 같습니다.

*   한국어 텍스트 데이터 전처리.
*   감성 분류 모델 학습.
*   새로운 텍스트에 대한 감성 예측.

## 코드 구조

프로젝트는 다음 파일들로 구성되어 있습니다:

*   `main.py`: 프로젝트의 메인 실행 파일입니다. 전처리된 데이터를 로드하고, 모델을 학습시키며, 감성 예측을 수행합니다.
*   `text_preprocessing.py`: 한국어 텍스트 전처리 기능을 담당합니다. 형태소 분석, 불용어 제거 등의 작업을 수행합니다.
*   `model_trainer.py`: 감성 분석 모델을 학습시키고 평가하는 기능을 제공합니다. `scikit-learn` 라이브러리를 활용합니다.

## 실행 방법

이 프로젝트를 실행하기 위해서는 먼저 필요한 라이브러리들을 설치해야 합니다.

### 1. 필수 라이브러리 설치

명령 프롬프트 또는 터미널에서 다음 명령을 실행하여 필요한 라이브러리들을 설치합니다:

```bash
pip install konlpy scikit-learn numpy pandas
```

### 2. 프로젝트 실행

모든 라이브러리가 설치되면 `main.py` 파일을 실행하여 감성 분석 과정을 확인합니다:

```bash
python main.py
```

실행 결과는 콘솔에 출력됩니다. 이는 모델 학습 결과와 몇 가지 예시 문장에 대한 감성 예측을 포함합니다.

## 사용 라이브러리

*   **Konlpy**: 한국어 텍스트 전처리를 위한 라이브러리입니다. 형태소 분석 및 토큰화에 사용됩니다.
*   **scikit-learn**: 머신러닝 모델 구축 및 평가를 위한 라이브러리입니다. 텍스트 벡터화와 분류 모델 학습에 사용됩니다.
*   **NumPy**: 수치 계산을 위한 파이썬 라이브러리입니다.
*   **Pandas**: 데이터 처리 및 분석을 위한 라이브러리입니다.

## 추가 정보

*   이 프로젝트는 예시 데이터셋을 사용하여 모델을 학습시킵니다. 실제 응용 프로그램에서는 더 크고 다양한 데이터셋을 사용해야 합니다.
*   모델의 성능을 향상시키기 위해 다양한 전처리 기법, 피처 엔지니어링, 그리고 다른 머신러닝 모델을 탐색할 수 있습니다.
*   Konlpy 사용을 위해서는 Java 런타임 환경(JRE)이 필요할 수 있습니다. Konlpy 설치 시 문제가 발생하면 JRE 설치 여부를 확인하십시오.
