# 텍스트 분석 프로젝트: 키워드 빈도 분석기

이 프로젝트는 주어진 텍스트 파일에서 단어들의 빈도를 분석하고, 가장 자주 등장하는 단어들을 추출하는 간단한 애플리케이션입니다. 텍스트 데이터에서 중요한 키워드를 식별하는 데 사용될 수 있으며, 자연어 처리(NLP)의 기초적인 예시입니다.

## 코드 구조

*   `Main.java`:
    *   프로그램의 진입점입니다.
    *   사용자로부터 텍스트 파일 경로를 입력받습니다.
    *   `FileUtil`을 사용하여 파일 내용을 읽어옵니다.
    *   `TextAnalyzer`를 사용하여 텍스트를 분석하고 단어 빈도를 계산합니다.
    *   분석된 결과를 화면에 출력합니다.

*   `TextAnalyzer.java`:
    *   주어진 텍스트를 소문자로 변환하고, 구두점 및 특수 문자를 제거하여 단어를 추출합니다.
    *   각 단어의 출현 빈도를 계산합니다.
    *   선택적으로 불용어(stop words, 예: "은", "는", "이", "가" 등)를 제거할 수 있는 기능을 포함합니다. (현재 구현에서는 간단화를 위해 불용어 제거는 포함하지 않음)
    *   가장 빈번하게 등장하는 단어들을 지정된 개수만큼 반환합니다.

*   `FileUtil.java`:
    *   파일에서 텍스트 내용을 읽어오는 정적(static) 메서드를 제공합니다.
    *   파일 입출력 과정에서 발생할 수 있는 예외를 처리합니다.

## 실행 방법

### 1. 컴파일

프로젝트 디렉터리(현재 디렉터리)에서 다음 명령어를 사용하여 자바 소스 파일들을 컴파일합니다.

```bash
javac Main.java TextAnalyzer.java FileUtil.java
```

### 2. 실행

컴파일이 완료되면, 다음 명령어를 사용하여 프로그램을 실행합니다. 프로그램 실행 시, 분석할 텍스트 파일의 경로를 인수로 전달해야 합니다. 예시에서는 `input.txt` 파일을 가정합니다.

```bash
java Main input.txt
```

**예시:**

`input.txt` 파일의 내용이 다음과 같다고 가정합니다:
```
안녕하세요. 이것은 텍스트 분석 예시입니다. 텍스트 분석은 매우 유용합니다.
```

위 명령어를 실행하면, 프로그램은 `input.txt`를 읽고 단어 빈도를 분석하여 결과를 출력할 것입니다.

## 활용 라이브러리

이 프로젝트는 별도의 외부 라이브러리 없이 표준 자바 JDK만을 활용합니다. `java.util.Map`, `java.util.List`, `java.util.Scanner` 등과 같은 기본 컬렉션 및 유틸리티 클래스를 사용하여 핵심 기능을 구현합니다. 이는 외부 의존성 없이 순수 자바 코드로 텍스트 분석의 기본 원리를 이해하는 데 도움이 됩니다.
