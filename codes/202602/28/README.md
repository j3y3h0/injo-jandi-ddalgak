# 매출 데이터 분석 유틸리티

이 프로젝트는 주어진 가상의 매출 데이터를 분석하여 유의미한 통계 정보를 도출하는 소규모 JavaScript 기반 콘솔 애플리케이션입니다. 산업에서 흔히 사용되는 데이터 처리 및 보고 알고리즘을 단순화하여 구현하였습니다.

## 주요 기능

*   **데이터 정제**: 입력된 매출 데이터에서 유효하지 않거나 누락된 정보를 처리합니다.
*   **총 매출 계산**: 전체 기간 동안의 총 매출을 계산합니다.
*   **평균 매출 계산**: 트랜잭션당 평균 매출을 계산합니다.
*   **기간별 매출 필터링**: 특정 날짜 범위 내의 매출 데이터를 필터링합니다.

## 코드 구조

*   `main.js`: 애플리케이션의 시작점입니다. 샘플 데이터를 로드하고, `dataProcessor` 및 `reporter` 모듈을 사용하여 데이터 처리 흐름을 제어합니다.
*   `dataProcessor.js`: 매출 데이터를 정제하고, 총 매출, 평균 매출, 기간별 필터링 등 핵심 분석 로직을 담당합니다.
*   `reporter.js`: `dataProcessor`에서 처리된 데이터를 기반으로 최종 분석 보고서를 생성하고 콘솔에 출력합니다.
*   `utils.js`: 날짜 포맷팅, 통화 포맷팅 등 범용적으로 사용될 유틸리티 함수들을 포함합니다.

## 실행 방법

이 프로젝트는 Node.js 환경에서 실행됩니다.

1.  Node.js가 설치되어 있는지 확인하십시오.
2.  터미널 또는 명령 프롬프트에서 프로젝트 디렉터리(`C:\server\gpt-code-diary\codes\202602\28`)로 이동합니다.
3.  다음 명령어를 실행하여 애플리케이션을 시작합니다:
    ```bash
    node main.js
    ```
4.  콘솔에 분석 결과 보고서가 출력될 것입니다.
