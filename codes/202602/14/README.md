# CSV 데이터 분석기

## 개요

이 프로젝트는 C#으로 구현된 간단한 CSV 파일 파서 및 통계 분석 프로그램이다. CSV 파일을 읽어 데이터를 파싱하고, 특정 수치 열에 대한 평균, 중앙값, 표준 편차와 같은 기본적인 통계 정보를 계산하는 기능을 제공한다. 데이터 분석의 기초적인 부분을 직접 구현해보는 목적으로 제작되었다.

## 주요 기능

*   **CSV 파일 파싱**: 쉼표로 구분된 CSV 파일을 읽어 헤더와 데이터를 구조화된 형태로 파싱한다.
*   **데이터 레코드 관리**: 파싱된 각 행을 `DataRecord` 객체로 표현하여 필드 접근을 용이하게 한다.
*   **기초 통계 계산**: 주어진 수치 데이터 목록에 대해 평균, 중앙값, 표준 편차를 계산한다.

## 코드 구조

*   `Program.cs`: 프로그램의 진입점이다. 샘플 CSV 파일을 생성하고, `CsvParser`를 사용하여 파일을 파싱한 후, `StatisticsCalculator`를 이용하여 통계 분석을 수행하는 과정을 보여준다.
*   `CsvParser.cs`: CSV 파일을 읽고 파싱하는 로직을 담당한다. 파일 경로를 받아 `DataRecord` 객체 목록을 반환한다.
*   `StatisticsCalculator.cs`: 수치 데이터 목록을 입력받아 평균, 중앙값, 표준 편차 등 통계량을 계산하는 메서드를 포함한다.
*   `DataRecord.cs`: CSV 파일의 한 행을 나타내는 데이터 모델이다. 헤더 이름을 통해 필드 값에 접근할 수 있도록 돕는다.
*   `CsvAnalyzer.csproj`: .NET 프로젝트 설정 파일이다.

## 실행 방법

1.  **프로젝트 빌드**: .NET SDK가 설치된 환경에서, 프로젝트 루트 디렉터리(이 `README.md` 파일이 있는 곳)에서 다음 명령어를 실행한다:
    ```bash
    dotnet build
    ```
2.  **프로젝트 실행**: 빌드가 성공적으로 완료되면 다음 명령어로 프로그램을 실행한다:
    ```bash
    dotnet run
    ```

    프로그램이 실행되면 `sample.csv` 파일이 자동으로 생성되고, 해당 파일의 데이터를 파싱하여 "Score" 열에 대한 통계 분석 결과를 콘솔에 출력할 것이다.

## 사용된 기술 및 알고리즘

*   **C# .NET 8**: 프로그램 구현에 사용된 언어 및 프레임워크이다.
*   **파일 I/O**: `System.IO` 네임스페이스를 활용하여 CSV 파일을 읽고 쓴다.
*   **문자열 처리**: `string.Split()` 메서드를 사용하여 CSV 행을 필드로 분리한다.
*   **컬렉션**: `System.Collections.Generic.List<T>` 및 `System.Collections.Generic.Dictionary<TKey, TValue>`를 사용하여 데이터를 효율적으로 관리한다.
*   **LINQ (Language Integrated Query)**: 데이터 집합에 대한 쿼리 및 통계 계산에 활용된다 (`.Average()`, `.OrderBy()`, `.Sum()`).
*   **기초 통계 알고리즘**: 평균, 중앙값, 표준 편차 계산 로직이 직접 구현되었다.
