# 텍스트 데이터 분석 및 요약 도구

## 프로젝트 소개
이 프로젝트는 JavaScript로 구현된 소규모 텍스트 데이터 분석 및 요약 도구입니다. 주어진 텍스트에서 단어 빈도를 분석하고, 기본적인 요약 기능을 제공합니다. 뉴스 기사, 문서 등 다양한 텍스트 데이터의 핵심 내용을 빠르게 파악하는 데 활용될 수 있습니다.

## 주요 기능
- 텍스트 정제: 구두점 제거, 소문자 변환 등 분석을 위한 텍스트 전처리
- 단어 빈도 분석: 텍스트 내에서 각 단어가 얼마나 자주 출현하는지 계산
- 기본적인 텍스트 요약: 텍스트의 처음 몇 문장을 추출하여 요약

## 프로젝트 구조
```
.
├── README.md
├── main.js
├── textProcessor.js
└── utils.js
```

### 파일 설명
- `README.md`: 프로젝트에 대한 설명, 사용 방법 등이 담겨 있는 문서입니다.
- `main.js`: 프로그램의 진입점입니다. 샘플 텍스트를 로드하고, `textProcessor.js`의 기능을 호출하여 분석 및 요약을 수행한 후 결과를 출력합니다.
- `textProcessor.js`: 텍스트 분석 및 요약의 핵심 로직을 담고 있습니다. 단어 빈도 계산, 요약 함수 등이 여기에 구현되어 있습니다.
- `utils.js`: 텍스트 정제와 같은 공통적으로 사용될 유틸리티 함수들을 모아 놓은 파일입니다.

## 사용 기술 및 알고리즘
- **JavaScript**: 프로젝트 구현을 위한 주요 언어입니다.
- **단어 빈도 분석 (Word Frequency Analysis)**: 텍스트 내 단어들의 출현 빈도를 계산하는 기본적인 통계 분석 기법입니다.
- **토큰화 (Tokenization)**: 텍스트를 개별 단어(토큰)로 분리하는 과정입니다.
- **텍스트 정제 (Text Cleaning)**: 분석의 정확도를 높이기 위해 불필요한 문자(구두점 등)를 제거하고 텍스트를 표준화하는 기법입니다.
- (참고) 실제 산업에서는 더 정교한 요약을 위해 TF-IDF(Term Frequency-Inverse Document Frequency), Latent Semantic Analysis(LSA) 또는 최신 딥러닝 기반의 자연어 처리(NLP) 모델(예: Transformers)이 활용될 수 있습니다. 본 프로젝트에서는 개념 이해를 위한 단순화된 요약 방식을 사용합니다.

2. 터미널을 열고 프로젝트 디렉터리로 이동합니다.

3. 다음 명령어를 실행하여 프로그램을 시작합니다.
   ```bash

   ```
