# CSV 데이터 처리 유틸리티 프로젝트

## 프로젝트 설명
이 프로젝트는 Node.js 환경에서 동작하는 간단한 CSV(Comma Separated Values) 데이터 처리 유틸리티입니다. CSV 파일로부터 데이터를 읽어 파싱하고, 특정 컬럼의 평균값을 계산하는 기능을 제공합니다. 소규모 데이터 분석이나 보고서 자동화 등 다양한 시나리오에서 활용될 수 있습니다.

## 코드 구조
*   `main.js`: 프로그램의 진입점입니다. CSV 파일 경로를 인자로 받아 파일 내용을 읽고, `csvParser.js`와 `dataProcessor.js`의 기능을 활용하여 데이터를 처리한 후 결과를 출력합니다.
*   `csvParser.js`: CSV 형식의 문자열을 파싱하여 JavaScript 객체 배열 형태로 변환하는 기능을 담당합니다. 헤더를 자동으로 인식하고 각 행의 데이터를 적절한 형식으로 변환합니다.
*   `dataProcessor.js`: 파싱된 데이터에 대한 실제 비즈니스 로직을 수행합니다. 현재는 특정 숫자 컬럼의 평균을 계산하는 기능을 포함하고 있습니다.
*   `utils.js`: 공통적으로 사용될 수 있는 유틸리티 함수들을 모아둔 파일입니다. 예를 들어, 간단한 로깅 함수 등을 포함할 수 있습니다.

## 실행 방법
1.  Node.js가 설치되어 있어야 합니다. (버전 12 이상 권장)
2.  처리하고자 하는 CSV 파일을 준비합니다. (예: `data.csv`)
    *   예시 `data.csv` 파일 내용:
        ```csv
        Name,Age,Score
        Alice,30,85
        Bob,24,92
        Charlie,35,78
        ```
3.  터미널 또는 명령 프롬프트에서 프로젝트 디렉터리로 이동합니다.
4.  다음 명령어를 실행하여 프로그램을 시작합니다:
    ```bash
    node main.js <CSV_파일_경로> <평균을_계산할_컬럼_이름>
    ```
    예시:
    ```bash
    node main.js data.csv Score
    ```

## 활용된 산업 표준 및 알고리즘
*   **CSV 파싱**: 이 프로젝트에서는 기본적인 문자열 분할 (`split`) 방식을 사용하여 CSV를 파싱합니다. 실제 산업 환경에서는 `csv-parser` (Node.js) 또는 `Papa Parse` (브라우저/Node.js)와 같은 전용 라이브러리를 사용하여 복잡한 케이스(콤마 포함 문자열, 인용 부호 처리 등)를 안전하게 처리하는 것이 일반적입니다.
*   **데이터 집계**: 평균 계산과 같은 기본적인 통계 알고리즘이 사용됩니다. 대규모 데이터셋의 경우, `Lodash`나 `Ramda`와 같은 유틸리티 라이브러리를 활용하면 더욱 효율적이고 가독성 높은 코드를 작성할 수 있습니다.

## 개발 노트
*   오류 처리가 최소화되어 있습니다. 실제 서비스에서는 파일 존재 여부, 파일 형식 유효성, 숫자 변환 실패 등에 대한 견고한 오류 처리가 필요합니다.
*   데이터 타입 변환은 현재 간단하게 구현되어 있습니다. 필요에 따라 더 복잡한 타입 유추 및 변환 로직을 추가할 수 있습니다.
