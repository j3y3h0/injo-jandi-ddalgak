# AI 기반 제조 공정 품질 검사 자동화 솔루션

이 프로젝트는 AI 기반 이미지 분석을 통해 제조 공정의 품질을 자동으로 검사하는 솔루션의 개념적 프로토타입입니다. 가상의 제조 라인에서 생산된 제품의 이미지를 분석하여 불량 여부를 판단하는 과정을 시뮬레이션합니다.

## 프로젝트 구조

-   `README.md`: 프로젝트에 대한 전반적인 설명, 구조 및 실행 방법을 담고 있습니다.
-   `main.py`: 프로그램의 진입점입니다. 이미지 처리 모듈과 품질 모델 모듈을 사용하여 전체 품질 검사 워크플로우를 조정합니다.
-   `image_processor.py`: 이미지 파일을 로드하고 전처리(예: 크기 조정, 그레이스케일 변환)하며, 품질 판단에 필요한 특징을 추출하는 기능을 담당합니다. `Pillow` 라이브러리를 활용합니다.
-   `quality_model.py`: 추출된 이미지 특징을 기반으로 제품의 품질(정상/불량)을 분류하는 가상의 AI 모델을 구현합니다. 실제 복잡한 AI 모델 대신 간단한 로직으로 분류를 시뮬레이션합니다.
-   `utils.py`: 파일 경로 관리, 가상 이미지 데이터 생성 등 프로젝트 전반에 걸쳐 사용될 유틸리티 함수들을 포함합니다.

## 실행 방법 (개념적)

이 프로젝트는 실제 AI 모델 훈련이나 복잡한 이미지 분석을 수행하지 않고, 그 과정을 시뮬레이션합니다. 코드를 실행하기 전에 필요한 라이브러리를 설치해야 합니다.

1.  **필요 라이브러리 설치 (가상):**
    ```bash
    pip install Pillow scikit-learn
    ```
    (참고: `scikit-learn`은 `quality_model.py`에서 개념적으로 사용되므로, 실제로는 복잡한 모델 없이도 실행 가능합니다.)

2.  **가상 이미지 데이터 준비:**
    `utils.py`에 포함된 `generate_dummy_image` 함수를 사용하여 검사에 사용될 가상의 이미지 파일들을 생성할 수 있습니다. 예를 들어, `main.py`에서 이 함수를 호출하여 `input_images` 디렉터리에 가상 이미지를 생성할 수 있습니다.

3.  **프로그램 실행:**
    `main.py` 파일을 실행하여 품질 검사 프로세스를 시작합니다.
    ```bash
    python main.py
    ```

    `main.py`는 `input_images` 디렉터리에 있는 이미지들을 하나씩 읽어 `image_processor.py`를 통해 특징을 추출하고, `quality_model.py`를 통해 품질을 예측한 후 결과를 출력할 것입니다.

## 기술 스택

-   **언어**: Python 3.x
-   **주요 라이브러리**:
    -   `Pillow`: 이미지 처리
    -   `scikit-learn`: (개념적으로) AI 모델 구성 및 분류

이 프로젝트는 AI 기반 품질 검사 자동화의 기본 흐름을 이해하는 데 도움을 주기 위한 예시입니다. 실제 산업 환경에서는 훨씬 복잡한 이미지 처리 기법과 딥러닝 모델이 활용됩니다.
