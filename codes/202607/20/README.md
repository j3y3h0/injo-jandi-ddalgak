# 텍스트 분석 시뮬레이터 (Text Analysis Simulator)

이 프로젝트는 "AI/머신러닝 기반 텍스트 분석"이라는 뉴스 주제를 바탕으로 구현된 소규모 C# 콘솔 애플리케이션입니다. 사용자로부터 텍스트를 입력받아 단어 빈도를 계산하고, 주요 구문을 식별하여 간단한 텍스트 분석 기능을 시뮬레이션합니다.

## 코드 구조

프로젝트는 다음 4개의 C# 파일로 구성되어 있습니다.

1.  **`Program.cs`**
    *   애플리케이션의 메인 진입점입니다.
    *   사용자로부터 텍스트 입력을 받습니다.
    *   `TextAnalyzer` 클래스의 `Analyze` 메서드를 호출하여 텍스트 분석을 수행합니다.
    *   분석 결과를 콘솔에 출력합니다.

2.  **`TextAnalyzer.cs`**
    *   텍스트 분석의 핵심 로직을 담당합니다.
    *   `Analyze` 메서드는 입력된 텍스트를 정제하고, 단어 빈도를 계산하며, 주요 구문을 식별합니다.
    *   내부적으로 `Utilities` 클래스의 텍스트 정제 기능을 활용합니다.
    *   `AnalysisResult` 객체를 반환합니다.

3.  **`AnalysisResult.cs`**
    *   텍스트 분석 결과를 담기 위한 데이터 구조를 정의합니다.
    *   `WordFrequencies` (단어와 그 빈도수를 저장하는 사전)와 `CommonPhrases` (주요 구문 목록) 속성을 포함합니다.

4.  **`Utilities.cs`**
    *   텍스트 처리와 관련된 유틸리티 함수를 제공하는 정적 클래스입니다.
    *   `CleanText` 메서드는 입력된 텍스트에서 구두점 및 특수 문자를 제거하고 소문자로 변환하는 기능을 수행합니다.

## 실행 방법

이 프로젝트는 .NET Core 또는 .NET Framework 환경에서 실행할 수 있습니다.

### 1. .NET SDK 설치

먼저, 시스템에 [.NET SDK](https://dotnet.microsoft.com/download)가 설치되어 있어야 합니다.

### 2. 프로젝트 디렉터리로 이동

터미널 또는 명령 프롬프트를 열고, 이 프로젝트 파일들이 있는 디렉터리로 이동합니다.
예시:
```bash
cd C:\server\gpt-code-diary\codes\202607\20
```

### 3. 프로젝트 실행

다음 명령어를 사용하여 애플리케이션을 빌드하고 실행할 수 있습니다.

```bash
dotnet run
```

애플리케이션이 시작되면, 텍스트를 입력하라는 메시지가 나타납니다. 텍스트를 입력하고 `Enter` 키를 누르면 분석 결과가 표시됩니다. `exit`를 입력하면 프로그램이 종료됩니다.
