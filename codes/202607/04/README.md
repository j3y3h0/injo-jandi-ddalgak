# JSON 데이터 처리 유틸리티

이 프로젝트는 JavaScript를 사용하여 JSON 데이터를 읽고, 필터링하고, 변환하며, 집계하는 간단한 유틸리티를 제공한다. 소규모 데이터 분석 및 처리 작업에 활용될 수 있다.

## 파일 구조

*   `README.md`: 이 문서이다.
*   `sample.json`: 예시로 사용될 원본 JSON 데이터 파일이다.
*   `utils.js`: JSON 파일 읽기/쓰기와 같은 보조 유틸리티 함수들을 포함한다.
*   `dataProcessor.js`: JSON 데이터를 필터링, 변환, 집계하는 핵심 로직을 포함한다.
*   `main.js`: `dataProcessor.js`와 `utils.js`를 활용하여 `sample.json`을 처리하고 그 결과를 콘솔에 출력하는 메인 스크립트이다.

## 기능

1.  **JSON 파일 읽기/쓰기**: `utils.js`를 통해 JSON 파일을 쉽게 읽고 쓸 수 있다.
2.  **데이터 필터링**: 특정 조건에 따라 데이터를 필터링할 수 있다.
3.  **데이터 변환**: 데이터의 각 항목에 대해 사용자 정의 변환 함수를 적용할 수 있다.
4.  **데이터 집계**: 특정 키를 기준으로 데이터를 그룹화하고 집계(예: 합계)할 수 있다.

## 실행 방법

1.  **프로젝트 디렉터리로 이동**:
    ```bash
    cd C:\server\gpt-code-diary\codes\202607\04
    ```

2.  **메인 스크립트 실행**:
    ```bash
    node main.js
    ```

    `main.js`를 실행하면 `sample.json` 파일의 데이터를 읽어 처리한 후, 그 결과를 콘솔에 출력할 것이다.
