# CSV 데이터 분석 프로젝트

## 프로젝트 개요
이 프로젝트는 C#을 사용하여 CSV 파일로부터 데이터를 읽어들이고, 기본적인 통계 분석(최소값, 최대값, 평균)을 수행한 후 그 결과를 콘솔에 출력하는 소규모 콘솔 애플리케이션이다. 산업에서 널리 사용되는 데이터 처리 및 분석의 기초적인 과정을 보여주기 위해 고안되었다.

## 파일 구조
- `Program.cs`: 애플리케이션의 진입점이다. `CsvProcessor`와 `DataAnalyzer`를 사용하여 데이터 처리 흐름을 관리한다.
- `CsvProcessor.cs`: CSV 파일에서 데이터를 읽어와 파싱하는 역할을 담당하는 유틸리티 클래스이다. 각 행을 문자열 리스트로 변환한다.
- `DataAnalyzer.cs`: 파싱된 숫자 데이터에 대해 기본적인 통계 분석을 수행하는 클래스이다. 최소값, 최대값, 평균을 계산하는 메서드를 제공한다.
- `data.csv`: 샘플 데이터가 포함된 CSV 파일이다. 이 파일을 통해 애플리케이션의 동작을 테스트할 수 있다.

## 실행 방법
1.  **프로젝트 빌드**: .NET SDK가 설치되어 있어야 한다. 프로젝트 디렉터리(`C:\server\gpt-code-diary\codes\202607\18`)로 이동하여 다음 명령어를 실행한다:
    ```bash
    dotnet build
    ```
2.  **애플리케이션 실행**: 빌드가 성공적으로 완료되면 다음 명령어를 사용하여 애플리케이션을 실행한다:
    ```bash
    dotnet run
    ```
    애플리케이션은 `data.csv` 파일을 읽고 분석 결과를 콘솔에 출력할 것이다.

## 코드 설명
각 C# 파일에는 상세한 한글 주석이 포함되어 있어 코드의 기능과 작동 방식을 이해하는 데 도움을 줄 것이다. 특히, `CsvProcessor`는 CSV 데이터를 효율적으로 읽기 위한 기본적인 로직을 구현하고 있으며, `DataAnalyzer`는 LINQ를 활용하여 데이터 집계 작업을 수행하는 예시를 보여준다. 이 프로젝트는 실용적인 시나리오에서 C#을 사용한 데이터 처리의 시작점이 될 수 있다.
