# 데이터 분석 및 시각화 프로젝트

## 프로젝트 소개
이 프로젝트는 샘플 CSV 파일에서 데이터를 로드하고, 기본적인 통계 분석을 수행한 후, 그 결과를 시각화하는 파이썬 애플리케이션입니다. `pandas` 라이브러리를 사용하여 데이터를 효율적으로 처리하고, `matplotlib`을 사용하여 시각적인 그래프를 생성합니다.

## 코드 구조
- `README.md`: 프로젝트에 대한 설명, 코드 구조, 설치 및 실행 방법을 담고 있습니다.
- `requirements.txt`: 프로젝트 실행에 필요한 파이썬 라이브러리 목록을 정의합니다.
- `data.csv`: 분석 및 시각화에 사용될 샘플 데이터 파일입니다.
- `data_loader.py`: CSV 파일로부터 데이터를 로드하는 기능을 담당합니다.
- `analyzer.py`: 로드된 데이터에 대한 통계 분석(예: 평균, 중앙값)을 수행하는 기능을 제공합니다.
- `visualizer.py`: 분석된 데이터를 바탕으로 그래프를 생성하고 저장하는 기능을 수행합니다.
- `main.py`: 프로젝트의 메인 실행 파일로, 데이터 로딩, 분석, 시각화 과정을 통합하여 실행합니다.

## 설치 방법
프로젝트를 실행하기 전에 필요한 라이브러리를 설치해야 합니다. 다음 명령어를 사용하여 설치할 수 있습니다.

```bash
pip install -r requirements.txt
```

## 실행 방법
모든 라이브러리가 설치된 후, `main.py` 파일을 실행하여 데이터 분석 및 시각화 과정을 시작할 수 있습니다.

```bash
python main.py
```

실행이 완료되면 `output_plot.png` 파일이 현재 디렉터리에 생성될 것입니다. 이 파일은 데이터의 시각화 결과를 보여줍니다.
