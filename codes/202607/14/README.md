# 데이터 통계 분석 프로그램

이 프로젝트는 주어진 숫자 목록에서 평균, 최소값, 최대값과 같은 기본적인 통계 데이터를 계산하는 간단한 Java 애플리케이션입니다. 산업에서 흔히 사용되는 데이터 분석의 기초적인 개념을 보여주기 위해 제작되었습니다.

## 파일 구성

*   `Main.java`: 프로그램의 메인 진입점입니다. 샘플 데이터를 정의하고 `DataProcessor`를 사용하여 통계를 계산한 후 결과를 콘솔에 출력합니다. 빈 목록에 대한 예외 처리도 포함되어 있습니다.
*   `DataProcessor.java`: 데이터 통계 계산을 담당하는 클래스입니다. `calculateAverage`, `findMin`, `findMax` 메서드를 포함하며, Java의 Stream API를 활용하여 효율적으로 데이터를 처리합니다. 입력 목록이 비어있을 경우를 대비하여 `Optional` 타입을 사용하여 안전하게 값을 반환합니다.

이 프로젝트를 실행하려면 다음 단계를 따르십시오.

1.  **컴파일:** 터미널을 열고 프로젝트 디렉터리(`C:\server\gpt-code-diary\codes\202607\14`)로 이동한 후, 다음 명령어를 사용하여 Java 소스 파일을 컴파일합니다.

