# 단어 빈도 분석기 (Word Frequency Counter)

이 프로젝트는 텍스트 파일에서 단어 빈도를 분석하고, 가장 자주 나타나는 단어들을 순위별로 보여주는 간단한 C# 콘솔 애플리케이션입니다. C#의 기본 기능을 활용하여 파일 입출력, 문자열 처리, 그리고 딕셔너리(Dictionary)를 이용한 데이터 집계 방법을 시연합니다.

## 코드 구조

프로젝트는 다음 파일들로 구성되어 있습니다:

-   `WordFrequencyCounter.csproj`: C# 프로젝트 파일로, 프로젝트의 빌드 설정 및 의존성을 정의합니다.
-   `Program.cs`: 애플리케이션의 진입점입니다. 명령줄 인수를 파싱하고, `FileProcessor`를 사용하여 파일을 읽고, `WordFrequencyAnalyzer`를 사용하여 단어 빈도를 분석한 후 결과를 출력합니다.
-   `FileProcessor.cs`: 파일 읽기/쓰기 작업을 담당하는 유틸리티 클래스입니다. 현재는 파일에서 줄 단위로 텍스트를 읽는 기능을 제공합니다. 대용량 파일 처리에 효율적인 `File.ReadLines`를 사용합니다.
-   `WordFrequencyAnalyzer.cs`: 텍스트 줄에서 단어들을 추출하고, 각 단어의 출현 빈도를 계산하는 핵심 로직을 포함합니다. 정규 표현식(`System.Text.RegularExpressions.Regex`)을 사용하여 단어를 토큰화하며, 모든 단어를 소문자로 변환하여 대소문자 구분을 없앱니다.

## 실행 방법

이 애플리케이션을 실행하려면 .NET 8.0 SDK 이상이 설치되어 있어야 합니다.

1.  **프로젝트 빌드**:
    현재 디렉터리(`C:\server\gpt-code-diary\codes\202607\23`)에서 다음 명령어를 실행하여 프로젝트를 빌드합니다.

    ```bash
    dotnet build
    ```

2.  **실행**:
    빌드가 성공하면, 생성된 실행 파일(`WordFrequencyCounter.dll`)을 사용하여 애플리케이션을 실행할 수 있습니다. 다음은 예시입니다.

    ```bash
    dotnet run -- <입력파일경로> [상위N개단어]
    ```

    -   `<입력파일경로>`: 분석할 텍스트 파일의 경로입니다. (필수)
    -   `[상위N개단어]`: (선택 사항) 결과로 볼 상위 N개의 단어 수를 지정합니다. 지정하지 않으면 기본값인 10이 사용됩니다.

    **예시**:
    현재 디렉터리에 `sample.txt` 파일이 있고, 그 파일의 상위 5개 단어를 분석하려면:

    ```bash
    dotnet run -- sample.txt 5
    ```

## 사용된 라이브러리 및 알고리즘

-   **.NET 8.0 SDK**: 애플리케이션 개발 프레임워크입니다.
-   **System.IO**: 파일 입출력 처리를 위한 네임스페이스입니다. `File.ReadLines`는 대용량 파일 처리 시 메모리 효율성을 높이는 산업 표준 방식입니다.
-   **System.Collections.Generic.Dictionary<TKey, TValue>**: 단어와 그 빈도를 저장하는 데 사용되는 핵심 데이터 구조입니다. O(1) 평균 시간 복잡도로 단어 빈도를 빠르게 조회하고 업데이트할 수 있습니다.
-   **System.Text.RegularExpressions.Regex**: 텍스트에서 단어를 추출하기 위한 정규 표현식 라이브러리입니다. 복잡한 패턴 매칭에 널리 사용됩니다.
-   **LINQ (Language Integrated Query)**: `OrderByDescending`, `Take` 등의 메서드를 사용하여 단어 빈도를 정렬하고 상위 N개 결과를 가져오는 데 사용됩니다. 데이터를 효율적으로 쿼리하고 조작하는 강력한 도구입니다.

이 프로젝트는 간단하지만, 실제 텍스트 분석 및 데이터 처리 애플리케이션의 기반이 될 수 있는 요소들을 포함하고 있습니다.
