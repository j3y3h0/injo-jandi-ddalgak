# CSV 데이터 처리 유틸리티 프로젝트

이 프로젝트는 지정된 CSV 파일에서 데이터를 읽어와 간단한 분석을 수행하는 C# 콘솔 애플리케이션이다. 현재 구현은 CSV 파일 내 특정 숫자 컬럼의 평균을 계산하는 기능을 포함하고 있다.

## 1. 코드 구조

*   `Program.cs`:
    *   애플리케이션의 진입점이다.
    *   `CsvProcessor`를 사용하여 CSV 파일을 처리하고 결과를 출력한다.
    *   예시 CSV 파일 경로를 지정하고, 처리할 컬럼의 이름을 정의한다.
*   `CsvProcessor.cs`:
    *   CSV 파일을 읽고 파싱하는 로직을 담당한다.
    *   제공된 CSV 파일 경로와 분석할 컬럼 이름을 받아 해당 컬럼의 평균 값을 계산한다.
    *   `System.IO` 및 LINQ(`System.Linq`)를 활용하여 파일 I/O 및 데이터 처리를 효율적으로 수행한다.
*   `DataRecord.cs`:
    *   CSV 파일의 각 행을 나타내는 간단한 데이터 모델 클래스이다.
    *   동적으로 컬럼 데이터를 저장할 수 있도록 `Dictionary<string, string>`을 사용한다.

## 2. 실행 방법

이 프로젝트를 실행하기 위한 단계는 다음과 같다.

1.  **프로젝트 빌드:**
    터미널에서 현재 프로젝트 디렉토리로 이동하여 다음 명령어를 실행하여 프로젝트를 빌드한다.

    ```bash
    dotnet build
    ```

2.  **CSV 파일 준비:**
    프로젝트의 실행 파일과 같은 디렉터리(`bin/Debug/netX.Y/` 또는 `bin/Release/netX.Y/` 등)에 `sample.csv`라는 이름의 CSV 파일을 생성한다. 이 파일은 최소한 하나의 숫자 컬럼을 포함해야 한다.

    예시 `sample.csv` 내용:
    ```csv
    Name,Age,Score
    Alice,30,95
    Bob,24,88
    Charlie,35,72
    David,29,65
    Eve,22,91
    ```

3.  **애플리케이션 실행:**
    빌드가 완료되고 `sample.csv` 파일이 준비되었으면, 다음 명령어로 애플리케이션을 실행한다.

    ```bash
    dotnet run
    ```
    또는 빌드된 실행 파일의 경로로 직접 이동하여 실행할 수도 있다.

    ```bash
    ./bin/Debug/netX.Y/YourProjectName.dll
    ```
    (여기서 `netX.Y`는 대상 프레임워크 버전이며 `YourProjectName`은 프로젝트 이름이다.)

## 3. 코드 변경 및 확장

*   `sample.csv` 파일의 내용을 변경하여 다양한 데이터를 테스트할 수 있다.
*   `Program.cs`에서 `columnNameToAnalyze` 변수의 값을 변경하여 분석할 컬럼을 지정할 수 있다.
*   `CsvProcessor.cs`에 추가적인 데이터 분석 기능(예: 최소/최대 값, 중앙값, 합계 등)을 구현하여 유틸리티를 확장할 수 있다.
*   오류 처리 로직(예: 파일이 없거나, 컬럼을 찾을 수 없는 경우)을 더욱 견고하게 만들 수 있다.
