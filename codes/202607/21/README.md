# 한국어 텍스트 감성 분석 프로젝트

## 1. 프로젝트 개요

이 프로젝트는 간단한 파이썬 코드를 사용하여 한국어 텍스트의 감성을 분석합니다. 뉴스 기사 댓글, 소셜 미디어 게시물 등 다양한 텍스트 데이터에 대해 긍정, 부정 또는 중립 감성을 판별하는 것을 목표로 합니다. 복잡한 자연어 처리 모델 대신, 미리 정의된 긍정/부정 단어 사전을 기반으로 감성 점수를 계산하는 규칙 기반 방식을 사용합니다.

더 정교한 감성 분석을 위해서는 KoNLPy(형태소 분석)나 Hugging Face `transformers` 라이브러리(사전 학습된 모델 활용)와 같은 고급 자연어 처리 라이브러리 및 기계 학습 모델을 사용하는 것이 일반적입니다. 이 프로젝트는 개념 이해를 위한 소규모 예제이다.

## 2. 파일 구조

프로젝트는 다음 세 가지 주요 파이썬 파일로 구성되어 있다:

*   `main.py`: 프로그램의 진입점이다. 사용자로부터 텍스트를 입력받아 감성 분석을 요청하고, 그 결과를 출력한다.
*   `sentiment_analyzer.py`: 감성 분석의 핵심 로직을 포함한다. 텍스트를 토큰화하고, `utils.py`에서 로드한 감성 사전을 기반으로 감성 점수를 계산하여 최종 감성을 결정한다.
*   `utils.py`: 감성 분석에 필요한 긍정 및 부정 단어 사전을 제공하는 유틸리티 함수를 포함한다. 현재는 코드 내에 단어 사전이 하드코딩되어 있다.

```
.
├── main.py
├── sentiment_analyzer.py
└── utils.py
└── README.md
```

## 3. 실행 방법

프로젝트를 실행하는 방법은 간단하다. 터미널 또는 명령 프롬프트에서 `main.py` 파일을 직접 실행한다.

```bash
python main.py
```

프로그램이 시작되면, 분석하고자 하는 한국어 텍스트를 입력하라는 메시지가 표시될 것이다. 텍스트를 입력하고 Enter 키를 누르면 해당 텍스트의 감성 분석 결과(긍정/부정/중립 및 감성 점수)가 출력된다. 프로그램을 종료하려면 `exit`를 입력한다.

## 4. 주요 기능

*   **한국어 텍스트 감성 분석**: 사용자가 입력한 한국어 텍스트의 감성(긍정, 부정, 중립)을 판별한다.
*   **규칙 기반 감성 점수**: 텍스트 내에 포함된 긍정/부정 단어의 수를 기반으로 감성 점수를 계산한다.
*   **간단한 구조**: 이해하고 확장하기 쉬운 모듈화된 코드 구조를 가지고 있다.

## 5. 향후 개선 방향

*   **감성 사전 확장**: 더 크고 다양한 긍정/부정 단어 사전을 구축하거나 외부 데이터를 활용하여 분석 정확도를 높일 수 있다.
*   **형태소 분석 도입**: `KoNLPy`와 같은 한국어 형태소 분석기를 사용하여 단어의 정확한 의미 단위를 추출하고, 분석의 정교함을 향상할 수 있다.
*   **머신러닝 모델 적용**: Naive Bayes, SVM, 딥러닝(RNN, Transformer) 등 머신러닝/딥러닝 모델을 훈련하여 감성 분석의 성능을 대폭 개선할 수 있다.
*   **API 연동**: 외부 감성 분석 API를 활용하거나, 자체 모델을 REST API로 제공하여 서비스에 통합할 수 있다.
