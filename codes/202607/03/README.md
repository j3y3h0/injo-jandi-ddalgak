# 소규모 데이터 분석 및 시각화 프로젝트

## 개요
이 프로젝트는 파이썬을 사용하여 간단한 CSV 파일로부터 데이터를 로드하고, 기본적인 통계 분석을 수행한 후 결과를 그래프로 시각화하는 예제이다. `pandas` 라이브러리를 사용하여 데이터를 효율적으로 다루고, `matplotlib`을 사용하여 시각적인 결과물을 생성한다.

## 파일 구조
- `README.md`: 프로젝트에 대한 설명 및 실행 방법을 담고 있는 문서이다. (현재 파일)
- `requirements.txt`: 프로젝트 실행에 필요한 파이썬 라이브러리 목록을 정의한다.
- `sample_data.csv`: 분석 및 시각화에 사용될 예제 데이터 파일이다.
- `data_loader.py`: CSV 파일로부터 데이터를 로드하는 기능을 담당하는 모듈이다.
- `visualizer.py`: 로드된 데이터를 기반으로 그래프를 생성하는 기능을 담당하는 모듈이다.
- `main.py`: 프로젝트의 메인 실행 파일로, 데이터 로드, 분석, 시각화 과정을 총괄한다.

## 실행 방법

### 1. 필수 라이브러리 설치
프로젝트를 실행하기 전에 필요한 파이썬 라이브러리들을 설치해야 한다. 다음 명령어를 사용하여 `requirements.txt`에 명시된 라이브러리들을 설치할 수 있다:

```bash
pip install -r requirements.txt
```

### 2. 프로젝트 실행
라이브러리 설치가 완료되면 `main.py` 파일을 실행하여 데이터 분석 및 시각화 과정을 시작할 수 있다:

```bash
python main.py
```

`main.py`를 실행하면 `sample_data.csv` 파일의 데이터를 로드하여 간단한 통계 정보를 출력하고, `data_visualization.png`라는 이름의 시각화 이미지를 생성한다.

## 사용 라이브러리
- `pandas`: 데이터 조작 및 분석을 위한 강력한 라이브러리이다.
- `matplotlib`: 다양한 형태의 정적, 애니메이션, 인터랙티브 시각화를 생성하기 위한 라이브러리이다.

## 데이터 형식
`sample_data.csv` 파일은 다음과 같은 형식으로 구성되어야 한다:
```csv
날짜,값
2023-01-01,10
2023-01-02,12
2023-01-03,15
2023-01-04,13
2023-01-05,18
```
`날짜` 열은 시계열 데이터의 날짜 정보를, `값` 열은 해당 날짜의 특정 측정값을 나타낸다.
