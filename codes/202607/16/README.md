# 데이터 분석 및 보고서 생성 프로젝트

## 개요
이 프로젝트는 Javascript를 사용하여 간단한 데이터(CSV 형식)를 로드하고, 통계적으로 분석한 후, 그 결과를 보고서 형태로 출력하는 소규모 애플리케이션입니다. 실제 산업에서 활용되는 데이터 파싱 라이브러리(예: PapaParse)와 시각화 라이브러리(예: Chart.js)의 개념을 반영하여 구현되었습니다.

## 프로젝트 구조

-   `package.json`: 프로젝트 메타데이터와 의존성 관리를 위한 파일입니다. `papaparse` 라이브러리를 의존성으로 명시하였습니다.
-   `main.js`: 애플리케이션의 주요 진입점입니다. `dataLoader`, `analyzer`, `reporter` 모듈을 사용하여 데이터 로드, 분석, 보고서 생성을 지휘합니다.
-   `dataLoader.js`: CSV 형식의 문자열 데이터를 파싱하여 Javascript 객체 배열로 변환하는 역할을 합니다. 실제 `papaparse`와 같은 라이브러리를 사용할 경우의 개념을 담고 있습니다.
-   `analyzer.js`: 로드된 데이터에 대해 평균, 최소값, 최대값 등의 기본적인 통계 분석을 수행하는 모듈입니다.
-   `reporter.js`: 분석된 결과를 보기 좋은 형태로 포맷팅하여 콘솔에 출력하는 보고서 생성 모듈입니다.

## 실행 방법

1.  **Node.js 설치**: 이 프로젝트를 실행하려면 시스템에 Node.js가 설치되어 있어야 합니다. (버전 12 이상 권장)
2.  **의존성 설치**: 프로젝트 디렉터리에서 다음 명령어를 실행하여 필요한 Node.js 패키지(예: `papaparse` - 현재는 `package.json`에만 명시되어 있고 실제 코드는 모듈 내에 간단하게 구현되어 있습니다)를 설치합니다.
    ```bash
    npm install
    ```
3.  **프로젝트 실행**: 다음 명령어를 실행하여 데이터 분석 애플리케이션을 시작합니다.
    ```bash
    npm start
    ```
    또는
    ```bash
    node main.js
    ```

위 명령어를 실행하면 `main.js`가 실행되어 샘플 데이터를 처리하고, 분석 결과를 콘솔에 출력하는 보고서를 생성합니다.

## 추가 고려 사항 (시각화)
이 프로젝트는 콘솔 기반의 보고서 출력에 중점을 두었지만, 만약 웹 환경에서 실행된다면 `Chart.js`와 같은 라이브러리를 활용하여 분석된 데이터를 시각적으로 표현할 수 있습니다. 예를 들어, `main.js`의 `Value` 데이터를 `Date`를 X축으로 하는 라인 차트로 시각화하여 데이터의 추이를 한눈에 볼 수 있을 것입니다.
