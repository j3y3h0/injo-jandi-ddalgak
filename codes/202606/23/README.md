# 로그 파일 분석기 (Log File Analyzer)

이 프로젝트는 지정된 로그 파일 (`sample.log`)을 읽고, 파싱하여, 로그 레벨별 통계 및 고유 소스별 통계를 생성하는 간단한 Java 애플리케이션입니다. 또한, 특정 로그 레벨 (예: ERROR)에 해당하는 로그 목록을 필터링하여 보여줍니다.

## 📁 프로젝트 구조

프로젝트는 다음 파일들로 구성되어 있습니다.

-   `Main.java`: 애플리케이션의 진입점입니다. `sample.log` 파일을 읽고, `LogParser`를 사용하여 로그 엔트리를 파싱한 후, `LogAnalyzer`를 이용해 분석을 수행하고 결과를 콘솔에 출력합니다.
-   `LogEntry.java`: 단일 로그 엔트리(로그 한 줄)를 객체로 표현하는 데이터 모델 클래스입니다. 타임스탬프, 로그 레벨, 소스, 메시지 정보를 포함합니다.
-   `LogParser.java`: 로그 문자열(텍스트)을 `LogEntry` 객체로 변환하는 정적 유틸리티 클래스입니다. 정규 표현식을 사용하여 로그 라인을 파싱합니다.
-   `LogAnalyzer.java`: `LogEntry` 객체들의 목록을 받아 다양한 분석을 수행하는 클래스입니다. 로그 레벨별 집계, 고유 소스별 집계, 특정 레벨 로그 필터링 등의 기능을 제공합니다.
-   `sample.log`: 분석에 사용될 예시 로그 파일입니다. `LogParser.java`에 정의된 형식에 맞춰 작성되어 있습니다.

## 🚀 실행 방법

이 애플리케이션을 컴파일하고 실행하려면 Java Development Kit (JDK) 8 이상이 설치되어 있어야 합니다.

1.  **컴파일:**
    프로젝트 디렉터리로 이동하여 다음 명령어를 실행하여 모든 `.java` 파일을 컴파일합니다.

    ```bash
    javac Main.java LogEntry.java LogParser.java LogAnalyzer.java
    ```

    또는 모든 자바 파일을 한 번에 컴파일하려면:

    ```bash
    javac *.java
    ```

2.  **실행:**
    컴파일이 완료되면 다음 명령어를 사용하여 애플리케이션을 실행합니다.

    ```bash
    java Main
    ```

    애플리케이션은 `sample.log` 파일을 읽어 분석 결과를 콘솔에 출력합니다.

## 💡 코드 설명

-   `LogEntry`: 로그의 각 구성 요소를 구조화하여 저장합니다.
-   `LogParser`: 정규 표현식 (`Pattern`과 `Matcher`)을 사용하여 복잡한 문자열에서 원하는 정보를 추출하는 방법을 보여줍니다. `LocalDateTime`을 사용하여 시간 정보를 안전하게 처리합니다.
-   `LogAnalyzer`: Java 8의 Stream API (`stream().collect(Collectors.groupingBy(...))`)를 활용하여 데이터를 효율적으로 그룹화하고 집계하는 방법을 시연합니다. 이는 대량의 데이터를 처리할 때 매우 유용하며, 현대 Java 개발에서 널리 사용되는 패턴입니다.
-   `Main`: `java.nio.file.Files.lines()`를 사용하여 파일을 스트림으로 읽어 메모리 효율적으로 대용량 파일을 처리할 수 있는 방법을 보여줍니다. 예외 처리 (`try-catch`)를 통해 파일I/O 오류를 관리합니다.

이 프로젝트는 실제 산업 환경에서 로그 분석 시스템을 구축할 때 활용될 수 있는 기본적인 데이터 처리 및 분석 기법을 담고 있습니다.
