# 간단한 규칙 기반 이상 거래 감지 시스템

## 프로젝트 개요

이 프로젝트는 금융 거래에서 발생할 수 있는 이상(사기성) 거래를 미리 정의된 규칙에 기반하여 감지하는 소규모 시스템이다. 특정 금액을 초과하는 거래를 사기성으로 분류하는 간단한 규칙을 구현하여, 규칙 기반 감지 시스템의 기본 원리를 보여준다. 본 시스템은 확장 가능한 구조를 가지며, 다양한 감지 규칙을 쉽게 추가하고 관리할 수 있도록 설계되었다.

## 주요 기능

*   **거래 객체 정의**: 거래 ID, 사용자 ID, 금액, 타임스탬프를 포함하는 `Transaction` 객체를 정의한다.
*   **규칙 인터페이스**: 모든 감지 규칙이 따라야 할 `Rule` 인터페이스를 제공한다.
*   **금액 임계값 규칙**: 특정 금액을 초과하는 거래를 감지하는 `AmountThresholdRule`을 구현한다.
*   **감지기 코어**: `FraudDetector`는 여러 규칙을 등록하고, 들어오는 거래에 대해 등록된 모든 규칙을 적용하여 사기성 여부를 판단한다.

## 기술 스택

*   **언어**: Java 8 이상
*   **빌드 도구**: (본 프로젝트에서는 별도의 빌드 도구를 사용하지 않으며, 기본적인 Java 컴파일 및 실행을 가정한다.)

## 프로젝트 구조

```
.
├── Main.java                 # 메인 실행 파일
├── Transaction.java          # 거래 객체 정의
├── Rule.java                 # 규칙 인터페이스 정의
├── AmountThresholdRule.java  # 금액 임계값 규칙 구현
└── FraudDetector.java        # 사기 감지 로직 및 규칙 관리
└── README.md                 # 프로젝트 설명 파일 (현재 파일)
```

## 실행 방법

본 프로젝트는 별도의 외부 라이브러리나 복잡한 설정 없이 표준 Java 개발 키트(JDK)만으로 컴파일하고 실행할 수 있다.

1.  **소스 코드 컴파일**:
    프로젝트 디렉터리(현재 디렉터리)로 이동한 후, 다음 명령어를 사용하여 모든 `.java` 파일을 컴파일한다.

    ```bash
    javac *.java
    ```

2.  **프로그램 실행**:
    컴파일이 완료되면, `Main` 클래스를 실행하여 이상 거래 감지 시스템을 시작한다.

    ```bash
    java Main
    ```

실행 결과로 콘솔에 가상의 거래 데이터에 대한 사기 감지 여부가 출력될 것이다. 출력되는 결과는 `FraudDetector`에 등록된 `AmountThresholdRule`에 따라 금액이 1000.00을 초과하는 거래를 사기성으로 표시할 것이다.
