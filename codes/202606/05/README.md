# 센서 데이터 이상 감지 프로젝트

이 프로젝트는 시뮬레이션된 센서 데이터에서 이상치(anomaly)를 감지하는 간단한 예제입니다. `Isolation Forest` 알고리즘을 사용하여 데이터의 정상 범주에서 벗어나는 패턴을 식별하며, 생성된 데이터를 시각적으로 확인하는 기능도 포함하고 있습니다.

## 1. 프로젝트 구조

프로젝트는 다음의 파이썬 파일들로 구성되어 있습니다.

-   `main.py`: 프로젝트의 메인 실행 파일입니다. 데이터 생성, 모델 학습, 이상 감지 및 결과 시각화의 전체 과정을 조율합니다.
-   `data_generator.py`: 시뮬레이션된 시계열 센서 데이터를 생성합니다. 이 데이터는 주기성과 노이즈를 포함하며, 미리 정의된 비율의 이상치를 포함하도록 설계되었습니다.
-   `anomaly_detector.py`: `scikit-learn` 라이브러리의 `IsolationForest` 모델을 캡슐화한 클래스를 포함합니다. 이 클래스는 모델 학습 및 이상치 예측 기능을 제공합니다.
-   `utils.py`: 데이터 시각화를 위한 `plot_sensor_data_with_anomalies` 함수와 데이터프레임을 CSV 파일로 저장하는 `save_dataframe_to_csv` 함수 등 유틸리티 기능을 제공합니다.

## 2. 사용된 주요 라이브러리

-   **pandas**: 데이터 처리 및 분석에 사용됩니다.
-   **numpy**: 수치 계산 및 데이터 생성에 사용됩니다.
-   **scikit-learn**: 머신러닝 알고리즘 (특히 IsolationForest) 구현에 사용됩니다.
-   **matplotlib**: 데이터 시각화 (그래프 생성)에 사용됩니다.

## 3. 실행 방법

프로젝트를 실행하기 전에 필요한 라이브러리들을 먼저 설치해야 합니다.

1.  **필요한 라이브러리 설치:**
    ```bash
    pip install pandas numpy scikit-learn matplotlib
    ```

2.  **프로젝트 실행:**
    프로젝트의 메인 스크립트인 `main.py`를 실행하여 전체 워크플로우를 시작합니다.
    ```bash
    python main.py
    ```

### 실행 과정:

-   `main.py`가 실행되면 `data_generator.py`를 통해 시뮬레이션된 센서 데이터가 생성됩니다.
-   생성된 데이터는 `simulated_sensor_data.csv` 파일로 저장됩니다.
-   `anomaly_detector.py`에 정의된 `AnomalyDetector` 클래스를 사용하여 `Isolation Forest` 모델을 학습합니다.
-   학습된 모델은 센서 데이터에서 이상치를 감지하고, 그 결과를 `anomaly_prediction` 컬럼에 추가합니다.
-   `utils.py`의 시각화 함수를 사용하여 센서 데이터와 감지된 이상치를 그래프로 표시합니다. 빨간색 점은 모델이 감지한 이상치를 나타내며, 초록색 X 표시는 데이터 생성 시 주입된 실제 이상치를 나타냅니다.
-   최종 결과 (데이터 및 감지된 이상치 정보)는 `anomaly_detection_results.csv` 파일로 저장됩니다.
-   콘솔에는 각 단계별 진행 상황과 최종 결과 요약이 출력됩니다.

이 프로젝트는 실제 센서 데이터 분석 및 이상 감지 시스템 구축의 기초적인 개념을 이해하는 데 도움이 될 것입니다.
