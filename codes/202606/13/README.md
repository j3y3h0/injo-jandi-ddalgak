# 간단한 뉴스 요약기 프로젝트

## 프로젝트 설명

이 프로젝트는 인공지능(AI) 관련 뉴스 기사를 자동으로 요약하는 간단한 Javascript 애플리케이션입니다. 텍스트 처리 및 단어 빈도 분석을 통해 기사의 핵심 문장을 추출하여 요약본을 생성합니다. 이는 대규모 텍스트 데이터에서 중요한 정보를 빠르게 파악해야 하는 산업 분야에서 유용하게 활용될 수 있는 기본적인 기술을 보여줍니다.

## 프로젝트 구조

*   `main.js`: 애플리케이션의 시작점입니다. 뉴스 데이터를 불러와 전처리하고, 요약 함수를 호출하여 결과를 출력합니다.
*   `newsData.js`: 요약할 예시 뉴스 기사 텍스트를 담고 있습니다. 실제 환경에서는 외부 API나 데이터베이스에서 뉴스를 가져오도록 확장될 수 있습니다.
*   `textProcessor.js`: 텍스트 전처리 로직을 담당합니다. 문장 분리, 단어 토큰화, 불필요한 문자 제거 등의 기능을 수행합니다.
*   `summarizer.js`: 핵심 요약 로직을 구현합니다. 단어 빈도를 계산하고, 이를 기반으로 기사에서 가장 중요한 문장들을 선별하여 요약문을 만듭니다.

## 실행 방법

이 프로젝트는 Node.js 환경에서 실행됩니다.

1.  Node.js가 설치되어 있는지 확인합니다.
2.  프로젝트 디렉터리로 이동합니다.
3.  다음 명령어를 터미널에 입력하여 실행합니다:

    ```bash
    node main.js
    ```

    실행하면 요약된 뉴스 내용이 콘솔에 출력될 것입니다.

## 사용 기술

*   Javascript (ES6+)
*   기본적인 자연어 처리(NLP) 개념 (토큰화, 단어 빈도 분석, 불용어 처리)
