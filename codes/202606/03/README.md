# 주식 가격 예측 프로젝트 (ML.NET)

이 프로젝트는 C#과 ML.NET을 사용하여 주식의 종가를 예측하는 간단한 예제입니다. 가상의 주식 데이터를 기반으로 선형 회귀 모델을 훈련하고, 이를 사용하여 새로운 데이터에 대한 예측을 수행합니다.

## 프로젝트 구조

*   `Program.cs`: 애플리케이션의 진입점입니다. 데이터 생성, 모델 훈련 및 예측 과정을 조율합니다.
*   `StockData.cs`: 주식 데이터의 입력 및 예측 결과에 사용될 데이터 모델을 정의합니다.
*   `DataProcessor.cs`: 모델 훈련에 사용될 가상의 주식 데이터를 생성하는 유틸리티 클래스입니다.
*   `StockPredictor.cs`: ML.NET을 사용하여 모델을 훈련하고 예측을 수행하는 핵심 로직을 포함합니다.

## 사용된 기술

*   **C#**: 주 프로그래밍 언어.
*   **ML.NET**: Microsoft의 오픈소스 머신러닝 프레임워크. 선형 회귀 모델을 구축하는 데 사용됩니다.

## 코드 설명

### `StockData.cs`

주식 데이터의 구조를 정의하는 클래스입니다. `StockFeature`는 모델 훈련에 사용될 특징들(예: 시가, 거래량)을, `StockPrediction`은 모델의 예측 결과(예: 예측된 종가)를 나타냅니다.

### `DataProcessor.cs`

`GetSampleData()` 메서드를 통해 훈련 및 예측에 사용될 가상의 주식 데이터를 생성합니다. 실제 환경에서는 이 부분이 데이터베이스, CSV 파일 또는 API에서 데이터를 불러오는 로직으로 대체될 수 있습니다.

### `StockPredictor.cs`

`MLContext`를 초기화하고, 데이터를 로드한 후, 데이터 변환 파이프라인을 정의합니다. 여기서는 `SdcaRegressionTrainer`를 사용하여 선형 회귀 모델을 훈련합니다. `Train()` 메서드는 모델을 훈련하고, `Predict()` 메서드는 훈련된 모델을 사용하여 새로운 주식 데이터에 대한 종가를 예측합니다.

### `Program.cs`

1.  `DataProcessor`를 사용하여 가상 주식 데이터를 생성합니다.
2.  `StockPredictor` 인스턴스를 생성합니다.
3.  생성된 데이터로 모델을 훈련합니다.
4.  새로운 예측 데이터를 준비하고, 훈련된 모델로 종가를 예측합니다.
5.  예측 결과를 콘솔에 출력합니다.

## 실행 방법

1.  .NET SDK가 설치되어 있는지 확인합니다. (`dotnet --version` 명령어로 확인 가능)
2.  이 프로젝트 폴더에서 터미널을 엽니다.
3.  다음 명령어를 실행하여 프로젝트를 빌드하고 실행합니다:
    ```bash
    dotnet run
    ```
4.  콘솔에 모델 훈련 및 예측 결과가 출력될 것입니다.

**참고:** 이 프로젝트는 교육 및 예시 목적으로 가상의 데이터를 사용합니다. 실제 주식 예측에는 더 복잡한 모델, 다양한 특징, 그리고 실제 시장 데이터가 필요합니다.
