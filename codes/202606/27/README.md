# 한국어 감성 분석기

## 프로젝트 설명
이 프로젝트는 간단한 규칙 기반의 한국어 텍스트 감성 분석기입니다. 미리 정의된 긍정 및 부정 키워드 목록을 기반으로 입력된 텍스트의 감성을 긍정, 부정, 또는 중립으로 분류합니다. 이는 자연어 처리(NLP) 분야의 기본적인 개념을 적용한 소규모 예제 프로젝트입니다.

## 파일 구성
- `README.md`: 현재 문서. 프로젝트에 대한 설명과 사용 방법을 제공합니다.
- `requirements.txt`: 이 프로젝트가 의존하는 Python 라이브러리 목록입니다.
- `utils.py`: 텍스트 전처리를 위한 유틸리티 함수들을 포함합니다.
- `sentiment_analyzer.py`: 핵심 감성 분석 로직을 구현한 모듈입니다.
- `main.py`: 프로그램의 진입점이며, 사용자로부터 텍스트를 입력받아 감성 분석을 수행하고 결과를 출력합니다.

## 실행 방법

### 1. 의존성 설치
이 프로젝트는 특별한 외부 라이브러리 없이 표준 Python 기능만을 사용하도록 설계되었지만, 향후 확장을 위해 `requirements.txt` 파일을 포함합니다. 현재는 추가 설치가 필요 없습니다.

### 2. 프로그램 실행
터미널에서 다음 명령어를 실행하여 프로그램을 시작합니다.
```bash
python main.py
```
프로그램이 시작되면 텍스트를 입력하라는 메시지가 나타납니다. 텍스트를 입력하고 엔터를 누르면 해당 텍스트의 감성 분석 결과가 출력됩니다.

## 주요 기능
- **한국어 텍스트 감성 분류**: 입력된 한국어 문장이 긍정적인지, 부정적인지, 혹은 중립적인지 판단합니다.
- **규칙 기반 분석**: 특정 키워드의 출현 빈도를 통해 감성을 유추하는 방식을 사용합니다.

## 개발 환경
- 언어: Python 3
- 운영체제: 독립적 (Windows, macOS, Linux 등)
