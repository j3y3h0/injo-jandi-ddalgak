# 주식 시장 예측 모델 프로젝트

## 1. 프로젝트 목표
이 프로젝트는 과거 주식 데이터를 기반으로 미래 주가 변동을 예측하는 간단한 머신러닝 모델을 구축하는 것을 목표로 합니다. `yfinance` 라이브러리를 사용하여 주식 데이터를 수집하고, `scikit-learn` 라이브러리의 `RandomForestRegressor` 모델을 사용하여 주식 가격을 예측합니다.

## 2. 파일 구조
프로젝트는 다음과 같은 파일들로 구성되어 있습니다.

-   `README.md`: 프로젝트에 대한 설명, 파일 구조 및 실행 방법을 담고 있는 문서입니다. (현재 파일)
-   `main.py`: 프로그램의 주 진입점입니다. 데이터 수집, 전처리, 모델 학습 및 예측 과정을 총괄합니다.
-   `data_fetcher.py`: 지정된 주식 티커와 기간에 대한 과거 주식 데이터를 Yahoo Finance에서 가져오는 기능을 제공합니다.
-   `preprocessor.py`: 가져온 주식 데이터를 머신러닝 모델 학습에 적합한 형태로 전처리합니다. 여기에는 기술적 지표(이동 평균 등) 계산 및 특성(Feature)과 타겟(Target) 분리가 포함됩니다.
-   `model_trainer.py`: 전처리된 데이터를 이용하여 `RandomForestRegressor` 모델을 학습시키고, 학습된 모델을 반환합니다.

## 3. 실행 방법

### 3.1. 필요한 라이브러리 설치
이 프로젝트를 실행하기 위해 다음 Python 라이브러리들이 필요합니다. `pip`를 사용하여 설치할 수 있습니다.

```bash
pip install pandas scikit-learn yfinance
```

### 3.2. 프로젝트 실행
라이브러리 설치가 완료되면, `main.py` 스크립트를 실행하여 주식 예측 모델을 작동시킬 수 있습니다.

```bash
python main.py
```

### 3.3. 결과 확인
`main.py` 실행이 완료되면, 모델의 예측 결과와 실제 값 비교 등 기본적인 결과가 콘솔에 출력될 것입니다. `main.py` 내의 `TICKER` 및 `START_DATE`, `END_DATE` 변수를 수정하여 다른 주식이나 다른 기간에 대한 예측을 시도해볼 수 있습니다.
