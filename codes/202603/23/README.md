# 주가 예측 미니 프로젝트

이 프로젝트는 자바를 사용하여 매우 단순한 이동 평균(Moving Average) 알고리즘으로 다음 날 주가를 예측하는 예제이다. 실제 주식 시장 예측은 훨씬 더 복잡한 모델과 방대한 데이터를 필요로 하지만, 이 프로젝트는 기본적인 알고리즘 적용과 자바 프로젝트 구성을 이해하는 데 초점을 맞추고 있다.

## 프로젝트 구조

*   `src/com/gemini/stockpredictor/`: 모든 자바 소스 파일이 위치한다.
    *   `StockData.java`: 개별 주식 데이터를 저장하는 클래스이다. 날짜와 주가를 포함한다.
    *   `StockPredictor.java`: 이동 평균 알고리즘을 구현하여 주가를 예측하는 핵심 로직을 담고 있다. 이 예제에서는 단순 평균을 사용한다.
    *   `Main.java`: 프로그램의 진입점으로, 예제 주식 데이터를 생성하고 `StockPredictor`를 사용하여 다음 날 주가를 예측한 후 결과를 출력한다.

## 실행 방법

이 프로젝트는 별도의 빌드 도구(Maven, Gradle 등) 없이 자바 표준 컴파일러(`javac`)와 자바 가상 머신(`java`)으로 직접 컴파일하고 실행할 수 있다.

1.  **소스 파일 컴파일**:
    프로젝트 루트 디렉터리(`codes/202603/23`)에서 다음 명령어를 실행하여 자바 소스 파일들을 컴파일한다.
    ```bash
    javac -d . src/com/gemini/stockpredictor/*.java
    ```
    이 명령어는 컴파일된 클래스 파일(`*.class`)들을 현재 디렉터리(프로젝트 루트)에 패키지 구조(`com/gemini/stockpredictor/`)에 맞게 생성한다.

2.  **프로그램 실행**:
    컴파일이 완료된 후, 프로젝트 루트 디렉터리에서 다음 명령어를 실행하여 프로그램을 실행한다.
    ```bash
    java com.gemini.stockpredictor.Main
    ```

3.  **예상 출력**:
    과거 주식 데이터와 예측된 다음 날 주가 정보가 콘솔에 출력될 것이다.

## 사용된 알고리즘 및 라이브러리

*   **알고리즘**: 단순 이동 평균 (Simple Moving Average - SMA)
    *   가장 기본적인 주가 예측 방법 중 하나로, 특정 기간 동안의 주가 평균을 계산하여 미래 주가를 예측하는 데 사용된다. 이 프로젝트에서는 모든 과거 데이터의 단순 평균을 사용한다.
*   **라이브러리**: 자바 표준 라이브러리 (java.util, java.lang)
    *   별도의 외부 라이브러리 없이 자바의 기본 기능을 활용하여 구현되었다. 이는 프로젝트의 단순성과 실행 용이성을 위한 선택이다.

## 한계점

이 프로젝트는 교육 및 예시 목적으로 매우 단순화된 모델을 사용한다. 실제 주가 예측은 시장의 복잡성, 다양한 경제 지표, 뉴스 이벤트, 투자자 심리 등 훨씬 더 많은 요소를 고려해야 하며, 통계학, 기계 학습 등의 고급 기술이 필수적이다.
