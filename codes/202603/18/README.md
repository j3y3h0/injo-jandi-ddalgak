# 주식 포트폴리오 분석기

이 프로젝트는 간단한 C# 콘솔 애플리케이션으로, 사용자가 보유한 주식 포트폴리오를 분석합니다. 주식 정보를 입력받아 총 자산 가치, 개별 주식의 손익률 등을 계산하여 보여줍니다.

## 코드 구조

*   **Program.cs**: 애플리케이션의 시작점입니다. 사용자로부터 주식 정보를 입력받고, `Portfolio` 클래스를 사용하여 포트폴리오를 관리하며, 분석 결과를 출력하는 역할을 합니다.
*   **Stock.cs**: 개별 주식의 정보를 정의하는 클래스입니다. 주식 코드, 이름, 매수 가격, 현재 가격, 수량 등의 속성을 포함합니다.
*   **Portfolio.cs**: 여러 `Stock` 객체를 관리하고 포트폴리오의 전체 가치 및 개별 주식의 손익률을 계산하는 로직을 담고 있습니다.
*   **FinancialAnalysis.cs**: 재무 관련 계산을 위한 유틸리티 클래스입니다. 예를 들어, 수익률 계산과 같은 기능을 제공합니다.
*   **StockPortfolioAnalyzer.csproj**: C# 프로젝트 파일로, 프로젝트의 구성 정보(타겟 프레임워크, 시작 객체 등)를 담고 있습니다.

## 실행 방법

1.  **Visual Studio 또는 .NET SDK 설치**: C# 프로젝트를 빌드하고 실행하려면 Visual Studio 또는 .NET SDK가 시스템에 설치되어 있어야 합니다.
2.  **프로젝트 디렉터리로 이동**: 명령 프롬프트나 터미널을 열고 프로젝트 디렉터리(`C:\server\gpt-code-diary\codes\202603\18`)로 이동합니다.
3.  **프로젝트 빌드**: 다음 명령어를 실행하여 프로젝트를 빌드합니다.
    ```bash
    dotnet build
    ```
4.  **애플리케이션 실행**: 빌드가 성공적으로 완료되면 다음 명령어를 사용하여 애플리케이션을 실행할 수 있습니다.
    ```bash
    dotnet run
    ```
5.  **사용**: 애플리케이션이 실행되면, 화면의 지시에 따라 주식 정보를 입력하고 포트폴리오 분석 결과를 확인할 수 있습니다.
