# 빅데이터 텍스트 분석기 프로젝트 (Big Data Text Analyzer Project)

## 프로젝트 개요

이 프로젝트는 빅데이터 처리 및 분석이라는 주제에 기반하여 텍스트 데이터에서 단어 빈도를 계산하고, 가장 빈번하게 나타나는 단어를 찾는 간단한 자바(Java) 애플리케이션입니다. 아파치 스파크(Apache Spark)와 같은 분산 처리 프레임워크에서 수행되는 워드 카운트(Word Count) 작업의 핵심 로직을 소규모로 구현하여, 기본적인 데이터 분석 과정을 이해하는 데 도움을 줍니다.

## 코드 구조 설명

프로젝트는 다음 세 가지 주요 자바(Java) 파일로 구성되어 있습니다.

1.  `MainApp.java`:
    *   이 애플리케이션의 시작점(Entry Point)입니다.
    *   샘플 텍스트 데이터를 준비하고, `WordCounter` 클래스를 사용하여 단어 빈도를 계산합니다.
    *   `DataUtil` 클래스를 활용하여 데이터를 전처리하고 결과를 출력하는 역할을 합니다.

2.  `WordCounter.java`:
    *   단어 빈도를 계산하는 핵심 로직을 포함하고 있습니다.
    *   주어진 단어 목록을 받아 각 단어의 출현 횟수를 `HashMap`을 이용하여 효율적으로 저장하고 관리합니다.
    *   이는 맵리듀스(MapReduce) 개념의 'Map' 단계와 'Reduce' 단계를 단순화한 형태입니다.

3.  `DataUtil.java`:
    *   데이터 처리와 관련된 유틸리티 메서드를 제공합니다.
    *   주어진 텍스트를 단어 단위로 분리하고, 불필요한 구두점이나 공백을 제거하는 전처리 기능을 포함합니다.
    *   이는 실제 빅데이터 처리에서 데이터 정제(Data Cleaning) 작업에 해당합니다.

## 실행 방법 (개념적 설명)

이 프로젝트는 표준 자바(Java) 개발 키트(JDK) 환경에서 컴파일하고 실행할 수 있습니다.

1.  **컴파일:**
    자바 컴파일러(javac)를 사용하여 `.java` 소스 파일들을 `.class` 파일로 컴파일합니다.
    ```bash
    javac -d out src/main/java/com/example/dataprocessor/*.java
    ```
    (여기서 `out`은 컴파일된 클래스 파일들이 저장될 디렉터리입니다.)

2.  **실행:**
    자바 가상 머신(JVM)을 사용하여 `MainApp` 클래스를 실행합니다.
    ```bash
    java -cp out com.example.dataprocessor.MainApp
    ```

위 명령어를 실행하면, `MainApp`에 정의된 샘플 텍스트 데이터를 처리하여 가장 빈번하게 나타나는 단어와 그 빈도수가 콘솔에 출력됩니다.

## 활용된 산업 라이브러리/알고리즘 (개념)

*   **맵리듀스(MapReduce) 개념:** `WordCounter`에서 단어 빈도를 계산하는 과정은 맵리듀스 프로그래밍 모델의 핵심 원리를 따릅니다. 텍스트를 단어로 분리하는 '맵(Map)' 단계와 각 단어의 수를 합산하는 '리듀스(Reduce)' 단계를 추상화하여 구현하였습니다.
*   **자바 표준 컬렉션:** `HashMap`과 같은 자바의 강력한 컬렉션 프레임워크를 활용하여 데이터를 효율적으로 저장하고 접근합니다. 이는 실제 빅데이터 처리 시스템의 내부에서도 자주 사용되는 기본적인 자료구조입니다.
*   **텍스트 전처리:** `DataUtil`에서 구현된 텍스트 전처리 로직은 자연어 처리(Natural Language Processing, NLP) 및 데이터 마이닝(Data Mining) 분야에서 필수적인 단계입니다.

이 프로젝트는 복잡한 빅데이터 시스템을 직접 구축하는 대신, 그 핵심적인 원리와 알고리즘을 자바 언어를 통해 간결하게 이해할 수 있도록 설계되었습니다.
