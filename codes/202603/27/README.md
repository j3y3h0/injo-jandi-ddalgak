# 실시간 주식 데이터 이상 감지 시스템 (Real-time Stock Data Anomaly Detection System)

## 프로젝트 소개

이 프로젝트는 실시간으로 유입되는 주식 가격 데이터를 시뮬레이션하고, 단순 이동 평균 및 표준 편차를 기반으로 잠재적인 가격 이상(급격한 상승 또는 하락)을 감지하는 소규모 C# 애플리케이션이다. 이는 금융 시장에서 비정상적인 거래 활동이나 중요한 시장 변화를 초기 단계에서 파악하는 데 활용될 수 있는 기초적인 아이디어를 제공한다.

## 프로젝트 구조

- `StockAnomalyDetection.csproj`: 프로젝트 파일로, .NET Core 빌드 설정을 포함한다.
- `Program.cs`: 애플리케이션의 진입점이다. 주식 데이터를 시뮬레이션하고, `AnomalyDetector`를 사용하여 이상 감지를 수행한 후 결과를 출력한다.
- `StockData.cs`: 개별 주식 거래 시점의 데이터를 나타내는 `StockTick` 클래스와, 주식 데이터 컬렉션을 관리하고 기본 통계 계산을 수행하는 `StockAnalyzer` 클래스를 정의한다.
- `AnomalyDetector.cs`: 지정된 기간 동안의 주식 가격 변동을 분석하여 이상 값을 감지하는 핵심 로직을 포함한다. 이동 평균과 표준 편차를 활용한다.

## 실행 방법

이 프로젝트는 .NET Core 환경에서 C#으로 작성되었다. 다음 단계를 따라 실행할 수 있다:

1.  **빌드**: 프로젝트 디렉터리에서 터미널을 열고 다음 명령어를 실행한다.
    ```bash
    dotnet build
    ```
2.  **실행**: 빌드가 성공적으로 완료되면 다음 명령어를 사용하여 애플리케이션을 실행한다.
    ```bash
    dotnet run
    ```

애플리케이션이 실행되면 시뮬레이션된 주식 데이터 스트림이 처리되고, 감지된 이상 현상이 콘솔에 출력될 것이다.

## 사용된 기술 및 알고리즘

-   **언어**: C#
-   **프레임워크**: .NET Core
-   **이상 감지 알고리즘**:
    *   **이동 평균 (Moving Average)**: 일정 기간 동안의 데이터 포인트 평균을 계산하여 데이터의 추세를 부드럽게 한다.
    *   **표준 편차 (Standard Deviation)**: 데이터 포인트가 평균으로부터 얼마나 퍼져 있는지를 측정하여 변동성을 파악한다.
    *   **Z-점수 (Z-score) 또는 임계값 기반 감지**: 현재 데이터 포인트가 이동 평균으로부터 표준 편차의 특정 배수 이상 벗어날 경우 이상 값으로 판단한다. 이 프로젝트에서는 단순 표준 편차 임계값을 사용한다.

이 프로젝트는 실제 시스템의 복잡성을 단순화한 예시이며, 실제 금융 시스템에서는 훨씬 더 정교한 알고리즘과 대규모 데이터 처리 기술이 사용된다.
