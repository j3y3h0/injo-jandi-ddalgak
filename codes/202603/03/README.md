# 간단한 데이터 분석 프로젝트

이 프로젝트는 Python의 Pandas 라이브러리를 사용하여 간단한 데이터 분석을 수행하는 예제입니다. 특정 CSV 파일로부터 데이터를 읽어와 기본적인 통계량(평균, 중앙값 등)을 계산하고 결과를 출력합니다.

## 프로젝트 구조

*   `main.py`: 프로젝트의 진입점입니다. 데이터를 로드하고, 분석 함수를 호출하며, 결과를 출력합니다.
*   `data_handler.py`: 데이터 로딩 및 전처리를 담당하는 유틸리티 파일입니다. CSV 파일을 Pandas DataFrame으로 읽어오는 함수를 포함합니다.
*   `analysis_functions.py`: 실제 데이터 분석 로직을 포함하는 유틸리티 파일입니다. 데이터프레임에 대한 통계 계산 함수를 제공합니다.
*   `sample_data.csv`: 분석에 사용될 샘플 데이터 파일입니다.

## 실행 방법

1.  **필요 라이브러리 설치**:
    프로젝트를 실행하기 전에 필요한 Python 라이브러리인 `pandas`를 설치해야 합니다. 다음 명령어를 사용하여 설치할 수 있습니다:
    ```bash
    pip install pandas
    ```

2.  **프로젝트 실행**:
    `main.py` 파일을 실행하여 데이터 분석을 시작합니다.
    ```bash
    python main.py
    ```

3.  **결과 확인**:
    스크립트가 실행되면, `sample_data.csv` 파일의 '값' 컬럼에 대한 평균과 중앙값이 콘솔에 출력될 것입니다.

## 코드 설명

각 Python 파일 내에는 주석으로 코드의 역할과 로직에 대한 상세한 설명이 포함되어 있습니다. Pandas 라이브러리의 기본적인 데이터 처리 및 분석 기능을 활용하는 방법을 보여줍니다.
