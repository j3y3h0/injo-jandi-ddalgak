# 텍스트 요약 프로젝트 (Javascript)

이 프로젝트는 자연어 처리(NLP) 기술 중 하나인 텍스트 요약을 자바스크립트로 구현한 소규모 예제입니다. 특정 주제(예: 뉴스 기사, 문서)의 핵심 내용을 빠르게 파악할 수 있도록 긴 텍스트에서 중요한 문장들을 추출하여 요약문을 생성합니다.

## 프로젝트 개요

이 요약기는 **추출 요약(Extractive Summarization)** 방식을 사용합니다. 추출 요약은 원문에서 중요한 문장들을 그대로 뽑아내어 요약문을 구성하는 방식입니다. 여기서 '중요도'는 문장 내의 단어들이 전체 텍스트에서 얼마나 자주 등장하는지(단어 빈도수)를 기반으로 평가됩니다.

산업 현장에서는 TF-IDF(Term Frequency-Inverse Document Frequency)나 TextRank와 같은 고급 알고리즘, 그리고 머신러닝 기반의 추상 요약(Abstractive Summarization) 모델(예: T5, BART 등)이 활용되지만, 이 프로젝트는 기본적인 원리를 자바스크립트 내장 기능으로 구현하여 요약 프로세스를 이해하는 데 초점을 맞추었습니다.

## 파일 구조

프로젝트는 다음 세 가지 주요 자바스크립트 파일과 README 파일로 구성되어 있습니다:

-   `main.js`: 프로그램의 진입점입니다. 예시 텍스트를 로드하고, `summarizer.js`의 요약 함수를 호출하여 결과를 출력합니다.
-   `textProcessor.js`: 텍스트를 문장과 단어로 분리하고, 불용어를 제거하는 등 기본적인 텍스트 전처리 기능을 담당합니다.
-   `summarizer.js`: `textProcessor.js`에서 제공하는 기능을 활용하여 텍스트의 단어 빈도수를 계산하고, 이를 바탕으로 각 문장의 중요도를 점수화하여 최종 요약문을 생성합니다.
-   `README.md`: 프로젝트에 대한 설명, 파일 구조, 그리고 실행 방법을 안내합니다.

## 실행 방법

이 프로젝트는 별도의 라이브러리 설치 없이 Node.js 환경에서 바로 실행할 수 있습니다.

1.  프로젝트 파일들이 있는 디렉터리로 이동합니다.
    ```bash
    cd C:\server\gpt-code-diary\codes\202603\11
    ```
2.  `main.js` 파일을 Node.js로 실행합니다.
    ```bash
    node main.js
    ```

실행하면 예시로 제공된 긴 텍스트의 원문과, 지정된 문장 수로 요약된 결과가 콘솔에 출력됩니다.
