# 판매 데이터 분석 프로젝트

## 프로젝트 설명

본 프로젝트는 가상의 판매 데이터를 C#으로 분석하고 요약 리포트를 생성하는 소규모 애플리케이션입니다. 각 제품별 총 판매액을 집계하여 콘솔에 출력하는 기능을 제공합니다.

## 코드 구조

프로젝트는 다음 파일들로 구성됩니다:

*   `SalesDataAggregation.csproj`: C# 프로젝트 파일입니다.
*   `Program.cs`: 애플리케이션의 진입점입니다. 판매 데이터 생성, 데이터 처리 및 리포트 생성을 조율합니다.
*   `SalesData.cs`: 개별 판매 기록을 나타내는 데이터 모델(클래스)을 정의합니다. 제품명, 수량, 단가 등의 정보를 포함합니다.
*   `DataProcessor.cs`: 판매 데이터를 받아 제품별 총 판매액을 집계하는 비즈니스 로직을 포함합니다.
*   `ReportGenerator.cs`: 집계된 판매 데이터를 기반으로 최종 리포트를 콘솔에 형식화하여 출력하는 기능을 담당합니다.

## 실행 방법

1.  **프로젝트 디렉터리로 이동**:
    명령 프롬프트 또는 터미널을 열고 프로젝트 디렉터리(현재 디렉터리)로 이동합니다.

2.  **애플리케이션 실행**:
    다음 명령어를 사용하여 애플리케이션을 실행할 수 있습니다:
    ```bash
    dotnet run
    ```
    (참고: `dotnet run` 명령은 프로젝트가 빌드되어 있지 않다면 자동으로 빌드한 후 실행합니다.)

## 사용된 기술 및 라이브러리 (또는 유사 기술)

본 프로젝트는 C# 언어의 기본 기능을 사용하여 데이터 집계를 수행합니다. 실제 산업 프로젝트에서는 더욱 복잡한 데이터 분석 및 통계 처리를 위해 [Math.NET Numerics](https://numerics.mathdotnet.com/)와 같은 라이브러리가 활용될 수 있습니다. 데이터 시각화가 필요한 경우 [OxyPlot](https://oxyplot.github.io/)과 같은 라이브러리도 고려해볼 수 있습니다. 본 예제에서는 기본적인 데이터 처리 로직을 통해 개념을 설명합니다.
