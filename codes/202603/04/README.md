# 뉴스 헤드라인 감성 분석 프로젝트

## 1. 프로젝트 개요

이 프로젝트는 Python과 `TextBlob` 라이브러리를 활용하여 간단한 뉴스 헤드라인 감성 분석을 수행하는 소규모 예제입니다. 각 뉴스 헤드라인이 긍정적인지, 부정적인지, 또는 중립적인지 판단하고, 해당 감성의 극성(polarity) 점수를 제공합니다. 이는 텍스트 데이터에서 전반적인 정서적 톤을 파악하는 데 유용합니다.

**참고:** `TextBlob` 라이브러리는 주로 영어 텍스트 분석에 최적화되어 있습니다. 따라서 이 프로젝트의 예제 헤드라인은 영어로 작성되었습니다. 한국어 텍스트의 정교한 감성 분석을 위해서는 `KoNLPy`와 같은 한국어 형태소 분석 라이브러리와 한국어 감성 분석을 위해 훈련된 모델(예: 딥러닝 기반 모델)을 함께 사용하는 것이 필요합니다.

## 2. 코드 구조

프로젝트는 다음 두 개의 Python 파일로 구성됩니다.

-   **`main.py`**:
    *   이 파일은 프로그램의 진입점입니다.
    *   분석할 샘플 뉴스 헤드라인 목록을 정의합니다.
    *   `sentiment_analyzer.py` 모듈에서 `SentimentAnalyzer` 클래스를 가져와 인스턴스를 생성합니다.
    *   각 헤드라인에 대해 감성 분석을 수행하고 그 결과를 콘솔에 출력합니다.

-   **`sentiment_analyzer.py`**:
    *   이 파일은 감성 분석 로직을 캡슐화하는 `SentimentAnalyzer` 클래스를 정의합니다.
    *   `TextBlob` 라이브러리를 사용하여 주어진 텍스트의 감성(긍정, 부정, 중립)과 극성 점수(polarity score)를 계산하는 `analyze_sentiment` 메서드를 제공합니다.

## 3. 실행 방법

프로젝트를 실행하려면 다음 단계를 따르십시오.

### 3.1. 필요한 라이브러리 설치

이 프로젝트는 `TextBlob` 라이브러리를 사용합니다. 다음 `pip` 명령어를 사용하여 설치할 수 있습니다.

```bash
pip install textblob
```

`TextBlob`은 내부적으로 자연어 처리 코퍼스(corpus)를 사용하므로, 다음 명령어를 실행하여 필요한 데이터를 다운로드해야 합니다.

```bash
python -m textblob.download_corpora
```

### 3.2. 프로젝트 실행

모든 종속성이 설치되면, `main.py` 파일을 실행하여 감성 분석을 시작할 수 있습니다.

```bash
python main.py
```

명령어를 실행하면 샘플 뉴스 헤드라인에 대한 감성 분석 결과가 콘솔에 출력될 것입니다.
