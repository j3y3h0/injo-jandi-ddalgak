# 뉴스 헤드라인 감성 분석기 (C#)

이 프로젝트는 C#으로 작성된 간단한 콘솔 애플리케이션으로, 입력된 뉴스 헤드라인이나 문장의 감성을 '긍정적', '부정적', '중립적'으로 분석합니다. 특정 키워드 목록을 기반으로 하는 사전 기반(Lexicon-based) 감성 분석 알고리즘을 사용합니다. 이는 머신러닝 모델 훈련 없이 빠르고 간단하게 감성 분석을 수행할 수 있는 방법론 중 하나입니다.

## 프로젝트 구조

- `NewsSentimentAnalyzer.csproj`: C# 프로젝트 파일입니다. .NET 8.0 콘솔 애플리케이션으로 구성되어 있습니다.
- `Program.cs`: 애플리케이션의 시작점입니다. 사용자로부터 텍스트를 입력받고, `SentimentAnalyzer` 클래스를 사용하여 감성을 분석한 후 결과를 출력합니다.
- `SentimentAnalyzer.cs`: 감성 분석의 핵심 로직을 포함하는 클래스입니다. 긍정 및 부정 단어 사전(lexicon)을 정의하고, 입력 텍스트를 분석하여 감성을 판단하는 `AnalyzeSentiment` 메서드를 제공합니다.

## 사용 언어 및 기술

- **언어**: C#
- **프레임워크**: .NET 8.0
- **알고리즘**: 사전 기반(Lexicon-based) 감성 분석

## 코드 설명

### `SentimentAnalyzer.cs`

-   `_positiveWords` 및 `_negativeWords`: 감성 분석에 사용될 긍정 및 부정 단어 목록입니다. 간단한 예제를 위해 수동으로 정의되었으며, 실제 환경에서는 훨씬 더 방대한 사전이 필요합니다.
-   `AnalyzeSentiment(string text)`:
    -   입력된 텍스트를 소문자로 변환하고 단어 단위로 분리합니다.
    -   분리된 각 단어가 긍정 또는 부정 단어 사전에 포함되어 있는지 확인하고 점수를 매깁니다.
    -   긍정 점수와 부정 점수를 비교하여 최종 감성(긍정적, 부정적, 중립적)을 반환합니다.

### `Program.cs`

-   `Main` 메서드는 `SentimentAnalyzer` 클래스의 인스턴스를 생성합니다.
-   무한 루프를 통해 사용자로부터 텍스트 입력을 지속적으로 받습니다.
-   입력된 텍스트를 `SentimentAnalyzer.AnalyzeSentiment` 메서드로 전달하여 감성을 분석합니다.
-   분석된 감성 결과를 콘솔에 출력합니다.
-   사용자가 'exit'를 입력하면 애플리케이션이 종료됩니다.

## 추가 고려사항

이 프로젝트는 매우 기본적인 감성 분석 예시입니다. 실제 감성 분석 시스템은 다음과 같은 점을 추가로 고려합니다.

-   **확장된 사전**: 훨씬 더 많은 긍정/부정 단어, 그리고 감성 강도를 나타내는 가중치를 포함하는 사전.
-   **부정어 처리**: "좋지 않다"와 같이 부정어가 포함된 문장의 감성을 올바르게 처리.
-   **문맥 이해**: 단어의 의미가 문맥에 따라 달라질 수 있음을 고려.
-   **머신러닝 모델**: 대량의 텍스트 데이터를 학습시켜 더욱 정확하고 복잡한 감성 패턴을 인식하는 머신러닝 또는 딥러닝 모델 사용 (예: ML.NET, BERT).
-   **이모티콘 및 신조어**: 온라인 텍스트에서 자주 사용되는 이모티콘, 신조어, 인터넷 용어 처리.
