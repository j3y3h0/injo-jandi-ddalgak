# 소규모 데이터 분석 프로젝트

## 프로젝트 개요

이 프로젝트는 간단한 CSV 파일로부터 데이터를 읽어와 기본적인 통계 분석(평균, 최솟값, 최댓값)을 수행하는 소규모 Java 애플리케이션입니다. 데이터 분석 및 시각화의 기초적인 개념을 이해하고, 실제 애플리케이션에서 데이터를 어떻게 처리하고 분석할 수 있는지 보여주는 예시입니다.

## 코드 구조

프로젝트는 다음 세 가지 주요 Java 파일로 구성되어 있습니다.

*   `Main.java`: 애플리케이션의 진입점입니다. `CsvParser`를 사용하여 `data.csv` 파일을 읽고, `StatisticsCalculator`를 사용하여 데이터를 분석한 후 결과를 콘솔에 출력합니다.
*   `CsvParser.java`: CSV 파일에서 데이터를 파싱하는 기능을 제공합니다. 파일 경로를 입력받아 각 줄을 읽고 쉼표(`,`)를 기준으로 문자열을 분리하여 숫자 데이터 목록으로 변환합니다.
*   `StatisticsCalculator.java`: 숫자 데이터 목록을 입력받아 평균, 최솟값, 최댓값을 계산하는 정적 메서드를 제공합니다.

또한, 분석에 사용될 샘플 데이터 `data.csv` 파일이 포함되어 있습니다.

## 실행 방법

이 프로젝트는 표준 Java 환경에서 컴파일하고 실행할 수 있습니다.

### 1. 컴파일

터미널 또는 명령 프롬프트를 열고 프로젝트의 루트 디렉터리(모든 `.java` 파일과 `data.csv`, `README.md` 파일이 있는 곳)로 이동합니다. 다음 명령어를 사용하여 Java 소스 파일을 컴파일합니다.

```bash
javac *.java
```

### 2. 실행

컴파일이 성공적으로 완료되면 다음 명령어를 사용하여 애플리케이션을 실행합니다.

```bash
java Main
```

애플리케이션은 `data.csv` 파일의 데이터를 분석하고, 그 결과를 콘솔에 출력할 것입니다.

## 데이터 형식

`data.csv` 파일은 쉼표로 구분된(CSV) 숫자 데이터를 포함해야 합니다. 각 줄은 하나의 숫자 값으로 구성되거나, 쉼표로 구분된 여러 숫자 값을 포함할 수 있습니다. 이 예제에서는 단순화를 위해 단일 열의 숫자 데이터를 가정합니다.

예시:
```
10,20,30,40,50
15,25,35
5,100,200,10
```
