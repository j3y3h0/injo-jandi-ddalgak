# 뉴스 기사 감성 분석 프로젝트

이 프로젝트는 간단한 C# 애플리케이션으로, 주어진 뉴스 기사의 텍스트를 분석하여 긍정 또는 부정 감성을 파악하는 것을 목표로 한다. 산업에서 널리 사용되는 데이터 처리 및 텍스트 분석의 기본 개념을 실용적인 형태로 구현하였다.

## 프로젝트 구조

*   `Program.cs`: 애플리케이션의 시작점이다. 뉴스 기사 데이터를 로드하고, 감성 분석을 수행한 후 결과를 출력하는 역할을 한다.
*   `NewsArticle.cs`: 뉴스 기사의 구조를 정의하는 클래스이다. 기사 제목과 내용을 속성으로 가진다.
*   `SentimentAnalyzer.cs`: 텍스트 감성 분석 로직을 포함하는 클래스이다. 간단한 키워드 매칭을 통해 감성을 판단한다.
*   `Utility.cs`: 파일 읽기/쓰기 등 범용적으로 사용될 유틸리티 함수들을 모아 놓은 클래스이다. (예: 임시 뉴스 데이터 파일 생성)

## 실행 방법

1.  **프로젝트 빌드**: .NET SDK가 설치되어 있다면, 프로젝트 루트 디렉터리(이 파일이 있는 곳)에서 다음 명령어를 실행하여 프로젝트를 빌드한다:
    ```bash
    dotnet build
    ```
2.  **실행**: 빌드가 성공하면 다음 명령어를 통해 애플리케이션을 실행한다:
    ```bash
    dotnet run
    ```

애플리케이션은 내부적으로 정의된 샘플 뉴스 데이터를 사용하여 감성 분석을 수행하고, 그 결과를 콘솔에 출력할 것이다.

## 주요 기능

*   샘플 뉴스 기사 데이터 로드.
*   뉴스 기사 내용 기반의 감성 분석 (긍정/부정).
*   분석 결과 콘솔 출력.

## 사용된 기술 및 라이브러리

*   C# 언어
*   .NET Core (콘솔 애플리케이션)
*   System.Text.Json (JSON 직렬화/역직렬화를 위한 내장 라이브러리)
