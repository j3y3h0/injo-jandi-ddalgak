# 뉴스 기사 감성 분석 파이썬 프로젝트

## 개요
이 프로젝트는 최근 AI 기술 동향에 대한 뉴스 기사의 감성을 분석하는 소규모 파이썬 프로젝트입니다. 텍스트 데이터를 전처리하고, 간단한 머신러닝 모델을 훈련하여 긍정/부정 감성을 분류합니다. 이 프로젝트는 `scikit-learn` 라이브러리를 활용하여 텍스트 분류의 기본적인 파이프라인을 보여줍니다.

## 파일 구조
- `main.py`: 프로그램의 주요 실행 파일입니다. 데이터 준비, 모델 훈련, 감성 분석 과정을 조율하고 시연합니다.
- `data_preprocessing.py`: 텍스트 데이터 전처리(불필요한 문자 제거, 소문자 변환) 및 토큰화 함수를 포함합니다.
- `model_training.py`: 감성 분석 모델 훈련 및 저장 기능을 담당합니다. 샘플 텍스트 데이터를 사용하여 TF-IDF 벡터라이저와 로지스틱 회귀 모델을 훈련합니다.
- `sentiment_analyzer.py`: 훈련된 모델을 로드하고, 새로운 텍스트의 감성(긍정 또는 부정)을 예측하는 기능을 제공합니다.

## 실행 방법

### 1. 필요 라이브러리 설치
프로젝트를 실행하기 전에 필요한 파이썬 라이브러리를 설치해야 합니다. 다음 명령어를 사용하여 설치할 수 있습니다.
```bash
pip install scikit-learn joblib
```

### 2. 프로젝트 실행
모든 라이브러리가 설치되었다면, `main.py` 파일을 실행하여 감성 분석 워크플로우를 시작할 수 있습니다.
```bash
python main.py
```

### 3. 예상 출력
`main.py`를 실행하면 모델 훈련 과정과 함께 몇 가지 샘플 뉴스 기사에 대한 감성 분석 결과가 콘솔에 출력됩니다. 이는 모델이 텍스트의 감성을 어떻게 분류하는지 보여줍니다.

## 추가 정보
이 프로젝트는 교육 및 시연 목적으로 간략화된 버전입니다. 실제 환경에서는 더 다양한 데이터셋, 복잡한 전처리 기법(예: 형태소 분석), 그리고 고급 머신러닝/딥러닝 모델을 사용하여 감성 분석의 정확도를 높일 수 있습니다. 모든 주석과 설명은 한글로 작성되었습니다.
