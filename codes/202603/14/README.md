# 간단한 데이터 처리 및 분석 프로젝트

## 프로젝트 개요
이 프로젝트는 CSV 파일에서 데이터를 읽어와 간단한 집계 및 분석을 수행하는 자바(Java) 애플리케이션입니다. '데이터 처리 및 분석'이라는 주제에 맞춰, 산업에서 흔히 사용되는 데이터 파싱, 집계, 보고서 생성의 기초적인 패턴을 구현하였습니다. 외부 라이브러리 없이 순수 자바(Java Standard Library)만을 사용하여 구현되어 있어, 기본적인 데이터 처리 로직을 이해하는 데 도움이 됩니다.

## 코드 구조
프로젝트는 다음과 같은 파일들로 구성되어 있습니다:

- `Main.java`: 애플리케이션의 진입점입니다. CSV 파일을 읽고, 데이터를 집계하며, 최종 보고서를 출력하는 전체 워크플로우를 담당합니다.
- `CsvProcessor.java`: CSV 파일에서 데이터를 읽고 파싱하는 기능을 제공합니다. 첫 번째 줄을 헤더로 인식하고, 각 데이터 행을 Map<String, String> 형태로 변환합니다.
- `DataAggregator.java`: 파싱된 데이터를 기반으로 특정 기준(예: 제품별, 지역별)에 따라 데이터를 집계하는 로직을 포함합니다.
- `ReportGenerator.java`: 집계된 데이터를 보기 좋게 콘솔에 출력하는 보고서 생성 기능을 담당합니다.
- `data.csv`: 애플리케이션이 처리할 샘플 CSV 데이터 파일입니다.

## 실행 방법

### 1. 전제 조건
자바 개발 키트(JDK) 8 이상이 설치되어 있어야 합니다.

### 2. 컴파일
프로젝트의 모든 `.java` 파일을 컴파일합니다. 현재 디렉터리에서 다음 명령어를 실행하십시오:
```bash
javac *.java
```
이 명령어는 `Main.java`, `CsvProcessor.java`, `DataAggregator.java`, `ReportGenerator.java` 파일을 컴파일하여 `.class` 파일을 생성합니다.

### 3. 실행
컴파일이 완료되면, `Main` 클래스를 실행하여 데이터 처리 및 분석 작업을 수행할 수 있습니다:
```bash
java Main
```
애플리케이션은 `data.csv` 파일을 읽고, 제품별 총 판매량과 지역별 평균 판매량을 계산하여 콘솔에 출력할 것입니다.

## 추가 고려 사항
- **외부 라이브러리**: 더 복잡한 CSV 처리나 데이터 분석을 위해서는 [Apache Commons CSV](https://commons.apache.org/proper/commons-csv/)나 [Eclipse Collections](https://www.eclipse.org/collections/)와 같은 산업 표준 라이브러리를 고려할 수 있습니다.
- **예외 처리**: 현재는 간단한 예외 처리만 포함되어 있습니다. 실제 상용 애플리케이션에서는 더 견고한 예외 처리 로직이 필요합니다.
- **데이터 유효성 검사**: CSV 파일의 데이터 유효성을 검사하는 로직을 추가하여 데이터의 신뢰성을 높일 수 있습니다.
