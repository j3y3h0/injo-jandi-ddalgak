# CSV 데이터 처리기 프로젝트

이 프로젝트는 C#을 사용하여 CSV(Comma Separated Values) 파일을 읽고, 데이터를 처리하며, 결과를 다시 CSV 파일로 저장하는 간단한 콘솔 애플리케이션입니다. 기본적인 파일 입출력 및 데이터 조작 기능을 보여주는 실용적인 예제입니다.

## 프로젝트 구조

*   `Program.cs`: 애플리케이션의 진입점입니다. `CsvProcessor` 클래스를 사용하여 CSV 파일을 읽고, 데이터를 처리하며, 결과를 출력하는 과정을 조율합니다.
*   `Person.cs`: CSV 파일의 각 행을 나타내는 데이터 모델 클래스입니다. `이름`, `나이`, `도시` 속성을 가집니다.
*   `CsvProcessor.cs`: CSV 파일을 읽고 쓰는 핵심 로직을 포함하는 유틸리티 클래스입니다. `List<Person>` 객체와 CSV 문자열 간의 변환을 담당합니다. 또한, 데이터를 필터링하고 평균 나이를 계산하는 등의 간단한 데이터 처리 기능을 제공합니다.
*   `DataGenerator.cs`: 샘플 CSV 데이터를 생성하여 파일로 저장하는 유틸리티 클래스입니다. 프로젝트 실행에 필요한 `sample_data.csv` 파일을 만듭니다.

## 실행 방법

1.  **프로젝트 빌드**: .NET SDK가 설치되어 있는 환경에서 프로젝트 루트 디렉터리(이 `README.md` 파일이 있는 곳)에서 다음 명령어를 실행하여 프로젝트를 빌드합니다.
    ```bash
    dotnet build
    ```
2.  **샘플 데이터 생성**: `DataGenerator.cs`를 사용하여 `sample_data.csv` 파일을 생성합니다. 애플리케이션을 처음 실행하면 `Program.cs`에서 자동으로 샘플 데이터를 생성하므로 이 단계를 수동으로 수행할 필요는 없습니다.
3.  **애플리케이션 실행**: 빌드가 성공하면 다음 명령어를 실행하여 애플리케이션을 실행합니다.
    ```bash
    dotnet run
    ```

애플리케이션은 `sample_data.csv` 파일을 읽어 데이터를 처리한 후, `filtered_data.csv` 및 `processed_data.csv` 파일을 생성할 것입니다. 콘솔에는 처리 결과 및 통계 정보가 출력됩니다.

## 주요 기능

*   **CSV 읽기**: CSV 파일에서 데이터를 읽어 `Person` 객체 리스트로 변환합니다.
*   **CSV 쓰기**: `Person` 객체 리스트를 CSV 형식의 파일로 저장합니다.
*   **데이터 필터링**: 특정 조건(예: 특정 도시)에 따라 데이터를 필터링합니다.
*   **데이터 요약**: 데이터의 평균 나이와 같은 통계 정보를 계산합니다.

## 사용된 기술

*   C#
*   .NET 8.0 (또는 그 이상)
*   `System.IO`: 파일 입출력을 위한 표준 라이브러리입니다.
*   `System.Linq`: 객체 컬렉션에 대한 쿼리 작업을 위한 표준 라이브러리입니다.
