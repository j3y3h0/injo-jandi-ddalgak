# 감성 분석 파이썬 프로젝트

이 프로젝트는 간단한 텍스트 감성 분석 도구입니다. 사용자로부터 입력받은 문장의 긍정적 또는 부정적 감성을 판별합니다.

## 프로젝트 구조

- `main.py`: 프로그램의 진입점입니다. 사용자로부터 텍스트를 입력받고, `sentiment_analyzer.py`의 기능을 호출하여 감성 분석을 수행한 후 결과를 출력합니다.
- `sentiment_analyzer.py`: 실제 감성 분석 로직을 포함합니다. 여기서는 미리 정의된 긍정/부정 키워드를 사용하여 감성을 분류하는 간단한 규칙 기반 방식을 사용합니다.
- `utils.py`: 텍스트 전처리(예: 토큰화)와 같은 유틸리티 함수를 제공합니다.

## 실행 방법

1.  **필요 라이브러리 설치 (선택 사항):**
    이 프로젝트는 파이썬 표준 라이브러리 외에 특별한 외부 라이브러리 설치가 필요하지 않습니다. (예시를 위해 간단한 규칙 기반으로 구현했기 때문입니다.)
    만약 더 복잡한 기능을 원한다면 `nltk`나 `scikit-learn` 같은 라이브러리를 설치할 수 있습니다.
    ```bash
    # 예시: NLTK 설치
    # pip install nltk
    ```

2.  **프로젝트 실행:**
    터미널을 열고 프로젝트 디렉터리로 이동한 후, 다음 명령어를 실행합니다.
    ```bash
    python main.py
    ```
    프로그램이 실행되면 텍스트를 입력하라는 메시지가 표시됩니다. 'exit'을 입력하면 프로그램이 종료됩니다.

## 코드 설명

### `main.py`

프로그램의 주요 흐름을 제어합니다. 사용자 입력을 받고, 감성 분석을 실행하며, 결과를 출력합니다.

### `sentiment_analyzer.py`

실제 감성 분석 로직이 구현된 파일입니다. 미리 정의된 긍정/부정 키워드 목록을 활용하여 입력 텍스트의 감성을 판단합니다.

### `utils.py`

텍스트 데이터를 감성 분석에 적합하게 가공하는 전처리 함수들을 포함합니다. 소문자 변환, 불필요한 문자 제거, 토큰화 등의 작업을 수행합니다.
