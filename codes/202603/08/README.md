# 한국어 뉴스 감성 분석 프로젝트

## 개요

이 프로젝트는 웹에서 한국어 뉴스 기사를 수집하고, 자연어 처리(NLP) 기술을 활용하여 기사의 감성을 분석하는 소규모 파이썬 프로젝트입니다. 특정 주제에 대한 대중의 감성 반응을 이해하는 데 도움을 줄 수 있습니다.

## 주요 기능

*   **뉴스 기사 수집**: 지정된 웹 페이지에서 뉴스 기사의 제목과 본문을 추출합니다.
*   **한국어 자연어 처리**: 형태소 분석을 통해 한국어 텍스트를 처리합니다.
*   **감성 분석**: 텍스트 내 긍정 또는 부정 키워드를 기반으로 기사의 감성 점수를 계산합니다.

## 프로젝트 구성

프로젝트는 다음 파일들로 구성되어 있습니다:

*   `main.py`: 프로젝트의 메인 실행 파일입니다. 뉴스 수집, 감성 분석 과정을 통합하여 실행하고 결과를 출력합니다.
*   `news_fetcher.py`: 웹 페이지에서 뉴스 기사(제목, 본문)를 안전하게 수집하는 기능을 담당합니다. `requests`와 `BeautifulSoup` 라이브러리를 사용합니다.
*   `nlp_processor.py`: 한국어 텍스트에 대한 자연어 처리(형태소 분석) 및 감성 분석을 수행합니다. `Konlpy` 라이브러리를 활용합니다.
*   `utils.py`: 텍스트 전처리(예: 불필요한 문자 제거) 및 파일 입출력 등 공통적으로 사용될 유틸리티 함수들을 포함합니다.

## 실행 방법

프로젝트를 실행하기 위한 단계는 다음과 같습니다.

### 1. 필수 라이브러리 설치

이 프로젝트는 `requests`, `beautifulsoup4`, `konlpy` 라이브러리를 사용합니다. 아래 명령어를 사용하여 설치하십시오:

```bash
pip install requests beautifulsoup4 konlpy
```

`konlpy` 설치 시 JDK (Java Development Kit)가 필요할 수 있습니다. JDK 설치 후 환경 변수 `JAVA_HOME`을 설정해야 합니다.

### 2. 프로젝트 실행

모든 라이브러리 설치가 완료되었다면, `main.py` 파일을 실행하여 뉴스 감성 분석을 시작할 수 있습니다.

```bash
python main.py
```

프로그램이 실행되면 예제 뉴스 URL에서 기사를 가져와 분석하고 결과를 터미널에 출력할 것입니다.

## 참고 사항

*   이 프로젝트의 감성 분석은 간단한 키워드 매칭 기반으로 구현되었습니다. 따라서 복잡한 문맥, 비유, 또는 아이러니 등을 정확하게 이해하지 못할 수 있습니다.
*   뉴스 기사 수집 시 웹사이트의 구조 변경에 따라 `news_fetcher.py`의 파싱 로직이 수정되어야 할 수도 있습니다.
*   `konlpy` 설치에 어려움이 있을 경우, [Konlpy 공식 문서](https://konlpy.org/ko/latest/install/)를 참고하십시오.
