# 텍스트 분석기 프로젝트 (Text Analyzer Project)

이 프로젝트는 주어진 텍스트를 분석하여 단어 빈도수와 총 단어 수를 계산하는 간단한 자바스크립트 애플리케이션이다. 텍스트 처리 및 기본 데이터 분석 기능을 제공한다.

## 파일 구조

*   `main.js`: 애플리케이션의 진입점이다. 입력 텍스트를 처리하고 `textProcessor.js`에 정의된 함수를 호출하여 분석을 수행한다. 최종 결과를 콘솔에 출력한다.
*   `textProcessor.js`: 텍스트 분석의 핵심 로직을 포함하고 있다. 단어 토큰화, 단어 빈도 계산 등의 기능을 제공한다.
*   `utils.js`: 텍스트 정제와 같은 유틸리티 함수들을 포함한다.

## 사용 언어

이 프로젝트는 순수 자바스크립트(JavaScript)로 작성되었다.

## 실행 방법

1.  프로젝트 디렉터리로 이동한다.
2.  Node.js가 설치되어 있어야 한다.
3.  터미널에서 다음 명령어를 실행한다:
    ```bash
    node main.js
    ```
4.  `main.js` 파일 내의 `sampleText` 변수 값을 수정하여 다른 텍스트를 분석할 수 있다.

## 예시

다음은 이 텍스트 분석기가 처리할 수 있는 작업의 예시이다:

*   문서 내에서 가장 자주 사용되는 단어 찾기
*   특정 키워드의 출현 빈도 확인
*   전체 단어 수 계산
